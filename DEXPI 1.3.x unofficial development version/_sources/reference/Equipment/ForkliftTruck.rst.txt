=============
ForkliftTruck
=============

Overview
========

.. rubric:: Class

A :doc:`MobileTransportSystem </reference/Equipment/MobileTransportSystem>` and vehicle with power operated prongs that can be raised and lowered by will, for loading, transporting and unloading goods (from `http://data.15926.org/rdl/RDS11590075 <http://data.15926.org/rdl/RDS11590075>`__).

.. image:: ForkliftTruck.*


.. rubric:: Supertypes

* :doc:`MobileTransportSystem </reference/Equipment/MobileTransportSystem>`

.. _dexpi-Equipment-ForkliftTruck-UpperLimitDischargeHead:

UpperLimitDischargeHead
=======================

.. rubric:: Attribute (data)


The upper limit for the discharge head of the :doc:`ForkliftTruck </reference/Equipment/ForkliftTruck>`.

Multiplicity
    0..1

Type
    :doc:`NullableLength </reference/PhysicalQuantities/NullableLength>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `UPPER LIMIT DISCHARGE HEAD <http://sandbox.dexpi.org/rdl/UpperLimitDischargeHead>`__

   Name
       UpperLimitDischargeHead

   AttributeURI
       `http://sandbox.dexpi.org/rdl/UpperLimitDischargeHead <http://sandbox.dexpi.org/rdl/UpperLimitDischargeHead>`__

.. admonition:: example

   The instance forkliftTruck1 represents a :doc:`ForkliftTruck </reference/Equipment/ForkliftTruck>` with an :ref:`UpperLimitDischargeHead <dexpi-Equipment-ForkliftTruck-UpperLimitDischargeHead>` of 37.0 cm.

   .. image:: /diagrams/diagram296.*


   .. admonition:: proteus-example

            <GenericAttribute
              Name="UpperLimitDischargeHead"

              Value="37.0"
              Units="Centimetre"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1318004" />
            ...
