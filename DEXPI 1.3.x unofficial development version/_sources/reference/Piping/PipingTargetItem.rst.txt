================
PipingTargetItem
================

Overview
========

.. rubric:: Abstract class

An item that can be the target of a :doc:`PipingConnection </reference/Piping/PipingConnection>` (attribute :ref:`TargetItem <dexpi-Piping-PipingConnection-TargetItem>`\ ) or a :doc:`PipingNetworkSegment </reference/Piping/PipingNetworkSegment>` (attribute :ref:`TargetItem <dexpi-Piping-PipingNetworkSegment-TargetItem>`\ ).

.. image:: PipingTargetItem.*


.. rubric:: Subtypes

* :doc:`FlowOutPipeOffPageConnector </reference/Piping/FlowOutPipeOffPageConnector>`

* :doc:`Nozzle </reference/Equipment/Nozzle>`

* :doc:`PipingComponent </reference/Piping/PipingComponent>`

* :doc:`PropertyBreak </reference/Piping/PropertyBreak>`

.. admonition:: proteus

   Implementation is subclass-specific.

.. admonition:: example

   As :doc:`PipingTargetItem </reference/Piping/PipingTargetItem>` is abstract, we consider :doc:`CheckValve </reference/Piping/CheckValve>` as an arbitrary concrete subclass.

   .. image:: /diagrams/diagram708.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml

            ID="checkValve1"
            ComponentClass="CheckValve"

        </PipingComponent>
