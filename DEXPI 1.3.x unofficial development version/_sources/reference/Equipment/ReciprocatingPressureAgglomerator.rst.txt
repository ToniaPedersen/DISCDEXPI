=================================
ReciprocatingPressureAgglomerator
=================================

Overview
========

.. rubric:: Class

An :doc:`Agglomerator </reference/Equipment/Agglomerator>` which uses pistons to produce pressure and to form material (from `http://data.15926.org/rdl/RDS2228720 <http://data.15926.org/rdl/RDS2228720>`__).

.. image:: ReciprocatingPressureAgglomerator.*


.. rubric:: Supertypes

Type
    :doc:`NullableForce </reference/PhysicalQuantities/NullableForce>`

   RDL reference
       `LOWER LIMIT DESIGN PRESSING FORCE <http://sandbox.dexpi.org/rdl/LowerLimitDesignPressingForce>`__

   Name

   AttributeURI
       `http://sandbox.dexpi.org/rdl/LowerLimitDesignPressingForce <http://sandbox.dexpi.org/rdl/LowerLimitDesignPressingForce>`__
