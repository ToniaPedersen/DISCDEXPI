=================
ContinuousWeigher
=================

Overview
========

.. rubric:: Class

A :doc:`Weigher </reference/Equipment/Weigher>` that weighs a mass flow rate in continuous mode.

.. image:: ContinuousWeigher.*


.. rubric:: Supertypes

* :doc:`Weigher </reference/Equipment/Weigher>`

.. _dexpi-Equipment-ContinuousWeigher-BeltWidth:

BeltWidth
=========

.. rubric:: Attribute (data)


The belt width of the :doc:`ContinuousWeigher </reference/Equipment/ContinuousWeigher>`.

Multiplicity
    0..1

Type
    :doc:`NullableLength </reference/PhysicalQuantities/NullableLength>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `BELT WIDTH <http://sandbox.dexpi.org/rdl/BeltWidth>`__

   Name
       BeltWidth

   AttributeURI
       `http://sandbox.dexpi.org/rdl/BeltWidth <http://sandbox.dexpi.org/rdl/BeltWidth>`__

.. admonition:: example

   The instance continuousWeigher1 represents a :doc:`ContinuousWeigher </reference/Equipment/ContinuousWeigher>` with a :ref:`BeltWidth <dexpi-Equipment-ContinuousWeigher-BeltWidth>` of 180.0 cm.

   .. image:: /diagrams/diagram168.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="continuousWeigher1"
            ComponentClass="ContinuousWeigher"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/ContinuousWeigher" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="BeltWidth"
              AttributeURI="http://sandbox.dexpi.org/rdl/BeltWidth"
              Format="double"
              Value="180.0"
              Units="Centimetre"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1318004" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
