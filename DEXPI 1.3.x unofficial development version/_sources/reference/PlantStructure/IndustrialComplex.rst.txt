.. _dexpi-PlantStructure-IndustrialComplex-ParentStructure:

ParentStructure
===============

.. rubric:: Attribute (reference)


A superordinate structure of which the :doc:`IndustrialComplex </reference/PlantStructure/IndustrialComplex>` is a part.

Multiplicity
    0..1

Type
    :doc:`IndustrialComplexParentStructure </reference/PlantStructure/IndustrialComplexParentStructure>`

Opposite multiplicity
    0..*

.. admonition:: proteus

   The attribute is implemented using :ref:`Proteus \<Association\> elements <proteus-concept-reference-property-association>`.

   Association type for the attribute ower
       ``"is a part of"``

   Opposite association type
       ``"is a collection including"``

.. admonition:: example

   .. image:: /diagrams/diagram732.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml

            ID="industrialComplex1"
            ComponentClass="IndustrialComplexIso102092012"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/IndustrialComplexIso102092012" DOTS123="DOTS123">
          ...
          <Association
            Type="is a part of"
            ItemID="enterprise1" />
          ...
        <PlantStructureItem />
        ...
        <PlantStructureItem
            ID="enterprise1"

          ...
          <Association
            Type="is a collection including"
            ItemID="industrialComplex1" />
          ...
        <PlantStructureItem />
