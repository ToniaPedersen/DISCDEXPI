.. _dexpi-Equipment-WasteGasEmitter-DesignVolumeFlowRate:

DesignVolumeFlowRate
====================

.. rubric:: Attribute (data)


The volume flow rate for which the :doc:`WasteGasEmitter </reference/Equipment/WasteGasEmitter>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullableVolumeFlowRate </reference/PhysicalQuantities/NullableVolumeFlowRate>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN VOLUME FLOW RATE <http://data.posccaesar.org/rdl/RDS14286227>`__

   Name
       DesignVolumeFlowRate

   AttributeURI
       `http://data.posccaesar.org/rdl/RDS14286227 <http://data.posccaesar.org/rdl/RDS14286227>`__

.. admonition:: example

   The instance wasteGasEmitter1 represents a :doc:`WasteGasEmitter </reference/Equipment/WasteGasEmitter>` with a :ref:`DesignVolumeFlowRate <dexpi-Equipment-WasteGasEmitter-DesignVolumeFlowRate>` of 420.0 m\ :sup:`3`/h.

   .. image:: /diagrams/diagram501.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="wasteGasEmitter1"
            ComponentClass="WasteGasEmitter"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/WasteGasEmitter" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="DesignVolumeFlowRate"
              AttributeURI="http://data.posccaesar.org/rdl/RDS14286227"
              Format="double"
              Value="420.0"
              Units="MetreCubedPerHour"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1321064" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
