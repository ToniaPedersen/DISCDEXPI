=============
NullableForce
=============

Overview
========

.. rubric:: Abstract data type

:doc:`NullableForce </reference/PhysicalQuantities/NullableForce>` is a :ref:`simple physical quantity type <dexpi-concept-simple-physical-quantity-type>` for the dimension LMT\ :sup:`-2`. :doc:`NullableForce </reference/PhysicalQuantities/NullableForce>` is abstract and has two concrete subtypes:

* a :doc:`Force </reference/PhysicalQuantities/Force>` is an :ref:`actual value <dexpi-concept-actual-value>` for a physical quantity with a numerical value and a unit of measurement;

* a :doc:`NullForce </reference/PhysicalQuantities/NullForce>` is a :ref:`null value <dexpi-concept-null-value>` that explicitly indicates the absence of an actual physical quantity.

.. image:: NullableForce.*


.. rubric:: Subtypes

* :doc:`Force </reference/PhysicalQuantities/Force>`

* :doc:`NullForce </reference/PhysicalQuantities/NullForce>`

.. admonition:: proteus

   All data attributes with type :doc:`NullableForce </reference/PhysicalQuantities/NullableForce>` are implemented as :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   See the examples for :doc:`Force </reference/PhysicalQuantities/Force>` and :doc:`NullForce </reference/PhysicalQuantities/NullForce>`.
