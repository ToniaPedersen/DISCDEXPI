===========================
NullableElectricalFrequency
===========================

Overview
========

.. rubric:: Abstract data type

A physical quantity of dimension T\ :sup:`-1` (inherited from :doc:`NullableFrequency </reference/PhysicalQuantities/NullableFrequency>`\ ) for the application type *electrical frequency*. :doc:`NullableElectricalFrequency </reference/PhysicalQuantities/NullableElectricalFrequency>` is abstract and has two concrete subtypes:

* an :doc:`ElectricalFrequency </reference/PhysicalQuantities/ElectricalFrequency>` is an :ref:`actual value <dexpi-concept-actual-value>` for a physical quantity with a numerical value and a unit of measurement;

* a :doc:`NullElectricalFrequency </reference/PhysicalQuantities/NullElectricalFrequency>` is a :ref:`null value <dexpi-concept-null-value>` that explicitly indicates the absence of an actual physical quantity.
