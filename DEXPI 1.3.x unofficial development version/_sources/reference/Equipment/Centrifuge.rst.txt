==========
Centrifuge
==========

Overview
========

.. rubric:: Class

A 'separator' and 'machine' that uses centrifugal force to separate phases of different densities (from `http://data.posccaesar.org/rdl/RDS420974 <http://data.posccaesar.org/rdl/RDS420974>`__).

.. image:: Centrifuge.*


.. rubric:: Supertypes

* :doc:`Equipment </reference/Equipment/Equipment>`

.. rubric:: Subtypes

* :doc:`CustomCentrifuge </reference/Equipment/CustomCentrifuge>`

* :doc:`FilteringCentrifuge </reference/Equipment/FilteringCentrifuge>`

* :doc:`SedimentalCentrifuge </reference/Equipment/SedimentalCentrifuge>`

DesignRotationalSpeed
=====================

.. rubric:: Attribute (data)

Multiplicity
    0..1

Type
    :doc:`NullableRotationalFrequency </reference/PhysicalQuantities/NullableRotationalFrequency>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN ROTATIONAL SPEED <http://sandbox.dexpi.org/rdl/DesignRotationalSpeed>`__

   Name
       DesignRotationalSpeed

   AttributeURI
       `http://sandbox.dexpi.org/rdl/DesignRotationalSpeed <http://sandbox.dexpi.org/rdl/DesignRotationalSpeed>`__

.. admonition:: example

   The instance centrifuge1 represents a :doc:`Centrifuge </reference/Equipment/Centrifuge>` with a :ref:`DesignRotationalSpeed <dexpi-Equipment-Centrifuge-DesignRotationalSpeed>` of 180.0 min\ :sup:`-1`.

   .. admonition:: proteus-example

          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="DesignRotationalSpeed"
              AttributeURI="http://sandbox.dexpi.org/rdl/DesignRotationalSpeed"
