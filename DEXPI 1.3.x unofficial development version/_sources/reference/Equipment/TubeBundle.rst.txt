==========
TubeBundle
==========

Overview
========

.. rubric:: Class

A bundle that consists of several tubes assembled together allowing multiple flow paths from a single source (from `http://data.posccaesar.org/rdl/RDS415259 <http://data.posccaesar.org/rdl/RDS415259>`__).

.. image:: TubeBundle.*

.. _dexpi-Equipment-TubeBundle-Chamber:

Chamber
=======

.. rubric:: Attribute (reference)


The :doc:`Chamber </reference/Equipment/Chamber>` in which the :doc:`TubeBundle </reference/Equipment/TubeBundle>` is located, if applicable. The Chamber must be a component of the same object as the TubeBundle.

Multiplicity
    0..1

Type
    :doc:`Chamber </reference/Equipment/Chamber>`

Opposite multiplicity
    0..*

.. admonition:: proteus

   The attribute is implemented using :ref:`Proteus \<Association\> elements <proteus-concept-reference-property-association>`.

   Association type for the attribute ower
       ``"is located in"``

   Opposite association type
       ``"is the location of"``

.. admonition:: example

   .. image:: /diagrams/diagram487.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="tubeBundle1"
            ComponentClass="TubeBundle"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS415259" DOTS123="DOTS123">
          ...
          <Association
            Type="is located in"
            ItemID="chamber1" />
          ...
        <Equipment />
        ...
        <Equipment
            ID="chamber1"
            ComponentClass="Chamber"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS903151421" DOTS123="DOTS123">
          ...
          <Association
            Type="is the location of"
            ItemID="tubeBundle1" />
          ...
        <Equipment />
