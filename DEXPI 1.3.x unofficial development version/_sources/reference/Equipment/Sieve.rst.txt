=====
Sieve
=====

Overview
========

.. rubric:: Class

A device that removes particles from a fluid when the fluid passes through or separates particles or molecules according to their size.

.. image:: Sieve.*

* :doc:`Equipment </reference/Equipment/Equipment>`

.. _dexpi-Equipment-Sieve-DesignMassFlowRate:

DesignMassFlowRate
==================

.. rubric:: Attribute (data)


The mass flow rate for which the :doc:`Sieve </reference/Equipment/Sieve>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullableMassFlowRate </reference/PhysicalQuantities/NullableMassFlowRate>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN MASS FLOW RATE <http://data.posccaesar.org/rdl/RDS14286182>`__

   Name
       DesignMassFlowRate

   AttributeURI
       `http://data.posccaesar.org/rdl/RDS14286182 <http://data.posccaesar.org/rdl/RDS14286182>`__

.. admonition:: example

   The instance sieve1 represents a :doc:`Sieve </reference/Equipment/Sieve>` with a :ref:`DesignMassFlowRate <dexpi-Equipment-Sieve-DesignMassFlowRate>` of 420.0 kg/s.

   .. image:: /diagrams/diagram451.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="sieve1"
            ComponentClass="Sieve"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/Sieve" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="DesignMassFlowRate"
              AttributeURI="http://data.posccaesar.org/rdl/RDS14286182"
              Format="double"
              Value="420.0"
              Units="KilogramPerSecond"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1329659" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
