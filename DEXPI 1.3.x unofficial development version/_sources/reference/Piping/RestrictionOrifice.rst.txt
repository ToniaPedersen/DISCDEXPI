==================
RestrictionOrifice
==================

Overview
========

.. rubric:: Class

A RESTRICTION ORIFICE is an ORIFICE PLATE that is intended for use as a restrictor.

.. image:: RestrictionOrifice.*


.. rubric:: Supertypes

* :doc:`PipeFitting </reference/Piping/PipeFitting>`

   The class is implemented as a :ref:`Proteus XML element with RDL reference <proteus-concept-xml-element-with-rdl>`.

   Tag
       ``<PipingComponent>``

   RDL reference
       `RESTRICTION ORIFICE <http://sandbox.dexpi.org/rdl/RestrictionOrifice>`__

   ComponentClass
       RestrictionOrifice

   ComponentClassURI
       `http://sandbox.dexpi.org/rdl/RestrictionOrifice <http://sandbox.dexpi.org/rdl/RestrictionOrifice>`__
