HeatTransferCoefficient

Overview
========

.. rubric:: Data type

An :ref:`actual value <dexpi-concept-actual-value>` for a physical quantity of type :doc:`NullableHeatTransferCoefficient </reference/PhysicalQuantities/NullableHeatTransferCoefficient>`, i.e., a physical quantity that has a numerical value and a unit of measurement.

.. admonition:: proteus

   All data attributes with type :doc:`NullableHeatTransferCoefficient </reference/PhysicalQuantities/NullableHeatTransferCoefficient>` (the base type of :doc:`HeatTransferCoefficient </reference/PhysicalQuantities/HeatTransferCoefficient>`\ ) are implemented as :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

.. admonition:: example

   The instance heatTransferCoefficient1 represents a :doc:`HeatTransferCoefficient </reference/PhysicalQuantities/HeatTransferCoefficient>` of 1.2 kW/(m\ :sup:`2`\Â·K).

   .. image:: /diagrams/diagram580.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <GenericAttribute
          Format="double"
          Value="1.2"
          Units="KilowattPerMetreSquaredKelvin"
          UnitsURI="http://data.posccaesar.org/rdl/RDS43167567170" DOTS123="DOTS123">
      

      Note that the ``<GenericAttribute>`` element must have a ``Name`` and an ``AttributeURI`` attribute. They depend on the data type attribute of the DEXPI class that owns the instance heatTransferCoefficient1. For a complete example, see the attribute :ref:`DesignHeatTransferCoefficient <dexpi-Equipment-HeatExchanger-DesignHeatTransferCoefficient>` of the DEXPI class :doc:`HeatExchanger </reference/Equipment/HeatExchanger>`.
