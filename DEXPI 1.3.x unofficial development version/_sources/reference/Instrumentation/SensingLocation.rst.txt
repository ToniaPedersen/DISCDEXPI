SensingLocation

Overview

.. rubric:: Abstract class

An object than can act as a :ref:`SensingLocation <dexpi-Instrumentation-ProcessSignalGeneratingFunction-SensingLocation>` of a :doc:`ProcessSignalGeneratingFunction </reference/Instrumentation/ProcessSignalGeneratingFunction>`.

.. image:: SensingLocation.*
