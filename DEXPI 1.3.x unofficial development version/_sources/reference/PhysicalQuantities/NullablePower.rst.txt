=============
NullablePower
=============

Overview
========

.. rubric:: Abstract data type

:doc:`NullablePower </reference/PhysicalQuantities/NullablePower>` is a :ref:`simple physical quantity type <dexpi-concept-simple-physical-quantity-type>` for the dimension L\ :sup:`2`\MT\ :sup:`-3`. :doc:`NullablePower </reference/PhysicalQuantities/NullablePower>` is abstract and has two concrete subtypes:

* a :doc:`Power </reference/PhysicalQuantities/Power>` is an :ref:`actual value <dexpi-concept-actual-value>` for a physical quantity with a numerical value and a unit of measurement;

* a :doc:`NullPower </reference/PhysicalQuantities/NullPower>` is a :ref:`null value <dexpi-concept-null-value>` that explicitly indicates the absence of an actual physical quantity.

.. image:: NullablePower.*


.. rubric:: Subtypes

* :doc:`NullPower </reference/PhysicalQuantities/NullPower>`

* :doc:`Power </reference/PhysicalQuantities/Power>`

.. admonition:: proteus

   All data attributes with type :doc:`NullablePower </reference/PhysicalQuantities/NullablePower>` are implemented as :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

.. admonition:: example

   See the examples for :doc:`Power </reference/PhysicalQuantities/Power>` and :doc:`NullPower </reference/PhysicalQuantities/NullPower>`.
