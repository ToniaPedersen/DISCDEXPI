===========================
ActuatingElectricalFunction
===========================

Overview
========

.. rubric:: Class

An actuation setting electrical function. It covers all types of electrical consumers, e.g., motors and heaters.

.. _dexpi-Instrumentation-ActuatingElectricalFunction-ActuatingElectricalFunctionNumber:

ActuatingElectricalFunctionNumber
=================================

.. rubric:: Attribute (data)


An identifier for the :doc:`ActuatingElectricalFunction </reference/Instrumentation/ActuatingElectricalFunction>`. It usually contains the identifier of the :doc:`ProcessInstrumentationFunction </reference/Instrumentation/ProcessInstrumentationFunction>` that includes the :doc:`ActuatingElectricalFunction </reference/Instrumentation/ActuatingElectricalFunction>` (see :ref:`ProcessInstrumentationFunctionNumber <dexpi-Instrumentation-ProcessInstrumentationFunction-ProcessInstrumentationFunctionNumber>`\ ).

Multiplicity
    0..1

Type
    :doc:`NullableString </reference/DataTypes/NullableString>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for string values <proteus-concept-dexpi-generic-attribute-string>`.

   RDL reference
       `ACTUATING ELECTRICAL FUNCTION NUMBER ASSIGNMENT CLASS <http://sandbox.dexpi.org/rdl/ActuatingElectricalFunctionNumberAssignmentClass>`__

   Name
       ActuatingElectricalFunctionNumberAssignmentClass

   AttributeURI
       `http://sandbox.dexpi.org/rdl/ActuatingElectricalFunctionNumberAssignmentClass <http://sandbox.dexpi.org/rdl/ActuatingElectricalFunctionNumberAssignmentClass>`__

.. admonition:: example

   "E4750.01" (:doc:`String </reference/DataTypes/String>`\ )

   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <ActuatingElectricalFunction
            ID="actuatingElectricalFunction1"
            ComponentClass="ActuatingElectricalFunction"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/ActuatingElectricalFunction" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
