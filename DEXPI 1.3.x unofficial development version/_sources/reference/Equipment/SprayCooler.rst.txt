===========
SprayCooler
===========

Overview
========

.. rubric:: Class

A :doc:`CoolingTower </reference/Equipment/CoolingTower>` that is based on spraying a coolant on a heated surface to be cooled.

.. image:: SprayCooler.*


.. rubric:: Supertypes

.. _dexpi-Equipment-SprayCooler-DesignSprayFlowRate:

DesignSprayFlowRate
===================

.. rubric:: Attribute (data)


The spray volume flow rate for the motive fluid for which the :doc:`SprayCooler </reference/Equipment/SprayCooler>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullableVolumeFlowRate </reference/PhysicalQuantities/NullableVolumeFlowRate>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN SPRAY FLOW RATE <http://sandbox.dexpi.org/rdl/DesignSprayFlowRate>`__

   Name
       DesignSprayFlowRate

   AttributeURI
       `http://sandbox.dexpi.org/rdl/DesignSprayFlowRate <http://sandbox.dexpi.org/rdl/DesignSprayFlowRate>`__

.. admonition:: example

   The instance sprayCooler1 represents a :doc:`SprayCooler </reference/Equipment/SprayCooler>` with a :ref:`DesignSprayFlowRate <dexpi-Equipment-SprayCooler-DesignSprayFlowRate>` of 40.0 m\ :sup:`3`/h.

   .. image:: /diagrams/diagram461.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="sprayCooler1"
            ComponentClass="SprayCooler"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/SprayCooler" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="DesignSprayFlowRate"
              AttributeURI="http://sandbox.dexpi.org/rdl/DesignSprayFlowRate"
              Format="double"
              Value="40.0"
              Units="MetreCubedPerHour"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1321064" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
