====
Area
====

Overview
========

.. rubric:: Data type

An :ref:`actual value <dexpi-concept-actual-value>` for a physical quantity of type :doc:`NullableArea </reference/PhysicalQuantities/NullableArea>`, i.e., a physical quantity that has a numerical value and a unit of measurement.

.. image:: Area.*


.. rubric:: Supertypes

* :doc:`NullableArea </reference/PhysicalQuantities/NullableArea>`

.. rubric:: Attributes (data)

+----------------------------------------------------------------------------+--------------+------------------------------------------------------------------------------------+
| Name                                                                       | Multiplicity | Type                                                                               |
+============================================================================+==============+====================================================================================+
| :ref:`Unit <dexpi-PhysicalQuantities-Area-Unit>`                           | 1            | :doc:`AreaUnit </reference/PhysicalQuantities/AreaUnit>`                           |
+----------------------------------------------------------------------------+--------------+------------------------------------------------------------------------------------+
| :ref:`Value <dexpi-PhysicalQuantities-Area-Value>`                         | 1            | :doc:`Double </reference/DataTypes/Double>`                                        |
+----------------------------------------------------------------------------+--------------+------------------------------------------------------------------------------------+

.. admonition:: proteus

   All data attributes with type :doc:`NullableArea </reference/PhysicalQuantities/NullableArea>` (the base type of :doc:`Area </reference/PhysicalQuantities/Area>`\ ) are implemented as :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

.. admonition:: example

   The instance area1 represents an :doc:`Area </reference/PhysicalQuantities/Area>` of 6.0 m\ :sup:`2`.

   .. image:: /diagrams/diagram574.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <GenericAttribute
          Format="double"
          Value="6.0"
          Units="MetreSquared"
          UnitsURI="http://data.posccaesar.org/rdl/RDS1358009" DOTS123="DOTS123">
      

      Note that the ``<GenericAttribute>`` element must have a ``Name`` and an ``AttributeURI`` attribute. They depend on the data type attribute of the DEXPI class that owns the instance area1. For a complete example, see the attribute :ref:`FilterArea <dexpi-Equipment-FilterUnit-FilterArea>` of the DEXPI class :doc:`FilterUnit </reference/Equipment/FilterUnit>`.
