==================
NullablePercentage
==================

Overview
========

.. rubric:: Abstract data type

A quantity given as a percentage. Although percentage is not a physical quantity type in the strict sense, it is modeled using the same pattern as for :ref:`simple physical quantity types <dexpi-concept-simple-physical-quantity-type>`. :doc:`NullablePercentage </reference/PhysicalQuantities/NullablePercentage>` is abstract and has two concrete subtypes:

* a :doc:`Percentage </reference/PhysicalQuantities/Percentage>` is an :ref:`actual value <dexpi-concept-actual-value>` for a quantity with a numerical value and a unit of measurement;

* a :doc:`NullPercentage </reference/PhysicalQuantities/NullPercentage>` is a :ref:`null value <dexpi-concept-null-value>` that explicitly indicates the absence of an actual quantity.
