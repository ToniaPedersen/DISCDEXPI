=====
Force
=====

Overview
========

.. rubric:: Data type

An :ref:`actual value <dexpi-concept-actual-value>` for a physical quantity of type :doc:`NullableForce </reference/PhysicalQuantities/NullableForce>`, i.e., a physical quantity that has a numerical value and a unit of measurement.

.. image:: Force.*


.. rubric:: Supertypes

* :doc:`NullableForce </reference/PhysicalQuantities/NullableForce>`

.. rubric:: Attributes (data)

+----------------------------------------------------------------------------+--------------+----------------------------------------------------------------------------------------+
| Name                                                                       | Multiplicity | Type                                                                                   |
+============================================================================+==============+========================================================================================+
| :ref:`Unit <dexpi-PhysicalQuantities-Force-Unit>`                          | 1            | :doc:`ForceUnit </reference/PhysicalQuantities/ForceUnit>`                             |
+----------------------------------------------------------------------------+--------------+----------------------------------------------------------------------------------------+
| :ref:`Value <dexpi-PhysicalQuantities-Force-Value>`                        | 1            | :doc:`Double </reference/DataTypes/Double>`                                            |
+----------------------------------------------------------------------------+--------------+----------------------------------------------------------------------------------------+

.. admonition:: proteus

   All data attributes with type :doc:`NullableForce </reference/PhysicalQuantities/NullableForce>` (the base type of :doc:`Force </reference/PhysicalQuantities/Force>`\ ) are implemented as :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

.. admonition:: example

   The instance force1 represents a :doc:`Force </reference/PhysicalQuantities/Force>` of 20.0 N.

   .. image:: /diagrams/diagram578.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <GenericAttribute
          Format="double"
          Value="20.0"
          Units="Newton"
          UnitsURI="http://data.posccaesar.org/rdl/RDS1337939" DOTS123="DOTS123">
      

      Note that the ``<GenericAttribute>`` element must have a ``Name`` and an ``AttributeURI`` attribute. They depend on the data type attribute of the DEXPI class that owns the instance force1. For a complete example, see the attribute :ref:`LowerLimitDesignPressingForce <dexpi-Equipment-ReciprocatingPressureAgglomerator-LowerLimitDesignPressingForce>` of the DEXPI class :doc:`ReciprocatingPressureAgglomerator </reference/Equipment/ReciprocatingPressureAgglomerator>`.
