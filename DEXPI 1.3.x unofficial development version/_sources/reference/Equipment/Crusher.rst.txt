=======
Crusher
=======

Overview
========

.. rubric:: Class

A mill that uses pressure or impact to reduce the particle size of solid materials (from `http://data.posccaesar.org/rdl/RDS11589940 <http://data.posccaesar.org/rdl/RDS11589940>`__).

.. image:: Crusher.*


.. rubric:: Supertypes

* :doc:`Mill </reference/Equipment/Mill>`

.. rubric:: Attributes (composition)

+------------------------------------------------------------------------------------------------+--------------+----------------------------------------------------------------------------------------+
| Name                                                                                           | Multiplicity | Type                                                                                   |
+================================================================================================+==============+========================================================================================+
| :ref:`CrusherElements <dexpi-Equipment-Crusher-CrusherElements>`                               | \*           | :doc:`CrusherElement </reference/Equipment/CrusherElement>`                            |
+------------------------------------------------------------------------------------------------+--------------+----------------------------------------------------------------------------------------+

.. admonition:: proteus

   The class is implemented as a :ref:`Proteus XML element with RDL reference <proteus-concept-xml-element-with-rdl>`.

   Tag
       ``<Equipment>``

   RDL reference
       `CRUSHER <http://data.posccaesar.org/rdl/RDS11589940>`__

   ComponentClass
       Crusher

   ComponentClassURI
       `http://data.posccaesar.org/rdl/RDS11589940 <http://data.posccaesar.org/rdl/RDS11589940>`__

.. admonition:: example

   .. image:: /diagrams/diagram181.*

.. _dexpi-Equipment-Crusher-CrusherElements:

CrusherElements
===============

.. rubric:: Attribute (composition)


The crusher elements of the :doc:`Crusher </reference/Equipment/Crusher>`.

Multiplicity
    \*

Type
    :doc:`CrusherElement </reference/Equipment/CrusherElement>`

Opposite multiplicity
    0..1

.. admonition:: proteus

   The attribute is implemented using the :ref:`XML hierarchy of the Proteus file <proteus-concept-composition-property-xml-child>`: The  element for the attribute value (a :doc:`CrusherElement </reference/Equipment/CrusherElement>`\ ) is a child of the ``<Equipment>`` element for the attribute owner (a :doc:`Crusher </reference/Equipment/Crusher>`\ ).

.. admonition:: example

   .. image:: /diagrams/diagram182.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="crusher1"
            ComponentClass="Crusher"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS11589940" DOTS123="DOTS123">
          ...
          <Equipment
              ID="crusherElement1"
              ComponentClass="CrusherUnit"
              ComponentClassURI="http://sandbox.dexpi.org/rdl/CrusherUnit" DOTS123="DOTS123">
            ...
          <Equipment />
          ...
        <Equipment />
