===================
ElectricalFrequency
===================

Overview
========

.. rubric:: Data type

An :ref:`actual value <dexpi-concept-actual-value>` for a physical quantity of type :doc:`NullableElectricalFrequency </reference/PhysicalQuantities/NullableElectricalFrequency>`, i.e., a physical quantity that has a numerical value and a unit of measurement.

.. image:: ElectricalFrequency.*


.. rubric:: Supertypes

* :doc:`NullableElectricalFrequency </reference/PhysicalQuantities/NullableElectricalFrequency>`

.. rubric:: Attributes (data)

+------------------------------------------------------------------------------------------------+--------------+----------------------------------------------------------------------------------------------------------------------------+
| Name                                                                                           | Multiplicity | Type                                                                                                                       |
+================================================================================================+==============+============================================================================================================================+
| :ref:`Unit <dexpi-PhysicalQuantities-ElectricalFrequency-Unit>`                                | 1            | :doc:`ElectricalFrequencyUnit </reference/PhysicalQuantities/ElectricalFrequencyUnit>`                                     |
+------------------------------------------------------------------------------------------------+--------------+----------------------------------------------------------------------------------------------------------------------------+
| :ref:`Value <dexpi-PhysicalQuantities-ElectricalFrequency-Value>`                              | 1            | :doc:`Double </reference/DataTypes/Double>`                                                                                |
+------------------------------------------------------------------------------------------------+--------------+----------------------------------------------------------------------------------------------------------------------------+

.. admonition:: proteus

   All data attributes with type :doc:`NullableElectricalFrequency </reference/PhysicalQuantities/NullableElectricalFrequency>` (the base type of :doc:`ElectricalFrequency </reference/PhysicalQuantities/ElectricalFrequency>`\ ) are implemented as :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

.. admonition:: example

   The instance electricalFrequency1 represents an :doc:`ElectricalFrequency </reference/PhysicalQuantities/ElectricalFrequency>` of 180.0 Hz.

   .. image:: /diagrams/diagram576.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <GenericAttribute
          Format="double"
          Value="180.0"
          Units="Hertz"
          UnitsURI="http://data.posccaesar.org/rdl/RDS1326464" DOTS123="DOTS123">
      

      Note that the ``<GenericAttribute>`` element must have a ``Name`` and an ``AttributeURI`` attribute. They depend on the data type attribute of the DEXPI class that owns the instance electricalFrequency1. For a complete example, see the attribute :ref:`AlternatingCurrentFrequency <dexpi-Equipment-AlternatingCurrentGenerator-AlternatingCurrentFrequency>` of the DEXPI class :doc:`AlternatingCurrentGenerator </reference/Equipment/AlternatingCurrentGenerator>`.
