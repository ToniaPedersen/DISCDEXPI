========================
PipingNetworkSegmentItem
========================

Overview
========

.. rubric:: Abstract class

An item that can be part of a :doc:`PipingNetworkSegment </reference/Piping/PipingNetworkSegment>`.

.. image:: PipingNetworkSegmentItem.*


.. rubric:: Subtypes

* :doc:`PipeOffPageConnector </reference/Piping/PipeOffPageConnector>`

* :doc:`PipingComponent </reference/Piping/PipingComponent>`

* :doc:`PropertyBreak </reference/Piping/PropertyBreak>`

.. admonition:: proteus

   Implementation is subclass-specific.

.. admonition:: example

   As :doc:`PipingNetworkSegmentItem </reference/Piping/PipingNetworkSegmentItem>` is abstract, we consider :doc:`CheckValve </reference/Piping/CheckValve>` as an arbitrary concrete subclass.

   .. image:: /diagrams/diagram699.*
