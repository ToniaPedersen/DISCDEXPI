=========
PlantArea
=========

Overview
========

.. rubric:: Class

An area as defined by ISA 95. The name PlantArea has been chosen to avoid confusion with the data type :doc:`Area </reference/PhysicalQuantities/Area>`.

.. image:: PlantArea.*


.. rubric:: Supertypes

* :doc:`PlantStructureItem </reference/PlantStructure/PlantStructureItem>`

.. _dexpi-PlantStructure-PlantArea-PlantAreaIdentificationCode:

PlantAreaIdentificationCode
===========================

.. rubric:: Attribute (data)


The identification code of the plant area according to ISA-95..

Multiplicity
    0..1

Type
    :doc:`NullableString </reference/DataTypes/NullableString>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for string values <proteus-concept-dexpi-generic-attribute-string>`.

   RDL reference
       `PLANT AREA IDENTIFICATION CODE ASSIGNMENT CLASS <http://sandbox.dexpi.org/rdl/PlantAreaIdentificationCodeAssignmentClass>`__

   Name
       PlantAreaIdentificationCodeAssignmentClass

   AttributeURI
       `http://sandbox.dexpi.org/rdl/PlantAreaIdentificationCodeAssignmentClass <http://sandbox.dexpi.org/rdl/PlantAreaIdentificationCodeAssignmentClass>`__

.. admonition:: example

   "F4" (:doc:`String </reference/DataTypes/String>`\ )

   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <PlantStructureItem
            ID="plantArea1"
            ComponentClass="AreaIsa95"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS10418236534" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="PlantAreaIdentificationCodeAssignmentClass"
              AttributeURI="http://sandbox.dexpi.org/rdl/PlantAreaIdentificationCodeAssignmentClass"
              Format="string"
              Value="F4" />
            ...
          </GenericAttributes>
          ...
        </PlantStructureItem>
