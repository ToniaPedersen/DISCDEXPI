==============
VibratingSieve
==============

Overview
========

.. rubric:: Class

A :doc:`Sieve </reference/Equipment/Sieve>` where the product to be sieved is transported over the mesh by vibration of the latter (from `http://data.15926.org/rdl/RDS2226670 <http://data.15926.org/rdl/RDS2226670>`__).

.. image:: VibratingSieve.*


.. rubric:: Supertypes

* :doc:`Sieve </reference/Equipment/Sieve>`

.. rubric:: Attributes (data)

+--------------------------------------------------------------------------------------------+--------------+------------------------------------------------------------------------------------------------+
| Name                                                                                       | Multiplicity | Type                                                                                           |
+============================================================================================+==============+================================================================================================+
| :ref:`DesignPower <dexpi-Equipment-VibratingSieve-DesignPower>`                            | 0..1         | :doc:`NullablePower </reference/PhysicalQuantities/NullablePower>`                             |
+--------------------------------------------------------------------------------------------+--------------+------------------------------------------------------------------------------------------------+

.. _dexpi-Equipment-VibratingSieve-DesignPower:

DesignPower
===========

.. rubric:: Attribute (data)


The power for which the :doc:`VibratingSieve </reference/Equipment/VibratingSieve>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullablePower </reference/PhysicalQuantities/NullablePower>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN POWER <http://sandbox.dexpi.org/rdl/DesignPower>`__

   Name
       DesignPower

   AttributeURI
       `http://sandbox.dexpi.org/rdl/DesignPower <http://sandbox.dexpi.org/rdl/DesignPower>`__

.. admonition:: example

   The instance vibratingSieve1 represents a :doc:`VibratingSieve </reference/Equipment/VibratingSieve>` with a :ref:`DesignPower <dexpi-Equipment-VibratingSieve-DesignPower>` of 500.0 kW.

   .. image:: /diagrams/diagram499.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="vibratingSieve1"
            ComponentClass="VibratingScreen"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/VibratingScreen" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="DesignPower"
              AttributeURI="http://sandbox.dexpi.org/rdl/DesignPower"
              Format="double"
              Value="500.0"
              Units="Kilowatt"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1330919" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
