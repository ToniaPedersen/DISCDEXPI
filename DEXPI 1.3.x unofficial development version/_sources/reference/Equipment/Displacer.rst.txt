.. _dexpi-Equipment-Displacer-Chamber:

Chamber
=======

.. rubric:: Attribute (reference)


The :doc:`Chamber </reference/Equipment/Chamber>` in which the :doc:`Displacer </reference/Equipment/Displacer>` is located, if applicable. The Chamber must be a component of the same object as the Displacer.

Multiplicity
    0..1

Type
    :doc:`Chamber </reference/Equipment/Chamber>`

Opposite multiplicity
    0..*

.. admonition:: proteus

   The attribute is implemented using :ref:`Proteus \<Association\> elements <proteus-concept-reference-property-association>`.

   Association type for the attribute ower
       ``"is located in"``

   Opposite association type
       ``"is the location of"``

.. admonition:: example

   .. image:: /diagrams/diagram234.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="displacer1"
            ComponentClass="Displacer"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/Displacer" DOTS123="DOTS123">
          ...
          <Association
            Type="is located in"
            ItemID="chamber1" />
          ...
        <Equipment />
        ...
        <Equipment
            ID="chamber1"
            ComponentClass="Chamber"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS903151421" DOTS123="DOTS123">
          ...
          <Association
            Type="is the location of"
            ItemID="displacer1" />
          ...
        <Equipment />

   AttributeURI
       `http://data.posccaesar.org/rdl/RDS1460719741 <http://data.posccaesar.org/rdl/RDS1460719741>`__
