===
Fan
===

Overview
========

.. rubric:: Class

An object that is capable of delivering or exhausting volumes of vapour or gas at low differential pressure (from `http://data.15926.org/rdl/RDS415169 <http://data.15926.org/rdl/RDS415169>`__).

.. image:: Fan.*


.. rubric:: Supertypes

* :doc:`Equipment </reference/Equipment/Equipment>`

.. rubric:: Subtypes

* :doc:`AxialFan </reference/Equipment/AxialFan>`

* :doc:`CustomFan </reference/Equipment/CustomFan>`

* :doc:`RadialFan </reference/Equipment/RadialFan>`

DesignDifferentialPressure
==========================

.. rubric:: Attribute (data)


The differential pressure for which the :doc:`Fan </reference/Equipment/Fan>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullablePressureAbsolute </reference/PhysicalQuantities/NullablePressureAbsolute>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN DIFFERENTIAL PRESSURE <http://sandbox.dexpi.org/rdl/DesignDifferentialPressure>`__

   Name
       DesignDifferentialPressure

   AttributeURI
       `http://sandbox.dexpi.org/rdl/DesignDifferentialPressure <http://sandbox.dexpi.org/rdl/DesignDifferentialPressure>`__

.. admonition:: example

   The instance fan1 represents a :doc:`Fan </reference/Equipment/Fan>` with a :ref:`DesignDifferentialPressure <dexpi-Equipment-Fan-DesignDifferentialPressure>` of 4.8 bar.

   .. image:: /diagrams/diagram270.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="fan1"
            ComponentClass="BlowerFan"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/BlowerFan" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="DesignDifferentialPressure"
              AttributeURI="http://sandbox.dexpi.org/rdl/DesignDifferentialPressure"
              Format="double"
              Value="4.8"
              Units="Bar"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1314539" />

          </GenericAttributes>
          ...
        </Equipment>
