.. _dexpi-Equipment-Vessel-NominalCapacity(Volume):

NominalCapacity(Volume)
=======================

.. rubric:: Attribute (data)


The nominal volumetric capacity of the :doc:`Vessel </reference/Equipment/Vessel>`.

Multiplicity
    0..1

Type
    :doc:`NullableVolume </reference/PhysicalQuantities/NullableVolume>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `NOMINAL CAPACITY VOLUME <http://sandbox.dexpi.org/rdl/NominalCapacityVolume>`__

   Name
       NominalCapacityVolume

   AttributeURI
       `http://sandbox.dexpi.org/rdl/NominalCapacityVolume <http://sandbox.dexpi.org/rdl/NominalCapacityVolume>`__

.. admonition:: example

   The instance vessel1 represents a :doc:`Vessel </reference/Equipment/Vessel>` with a :ref:`NominalCapacity(Volume) <dexpi-Equipment-Vessel-NominalCapacity(Volume)>` of 7.2 m\ :sup:`3`.

   .. image:: /diagrams/diagram497.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="vessel1"
            ComponentClass="Vessel"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS414674" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="NominalCapacityVolume"
              AttributeURI="http://sandbox.dexpi.org/rdl/NominalCapacityVolume"
              Format="double"
              Value="7.2"
              Units="MetreCubed"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1349099" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
