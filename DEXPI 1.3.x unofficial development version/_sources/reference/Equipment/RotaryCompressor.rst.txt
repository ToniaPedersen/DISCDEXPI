================
RotaryCompressor
================

Overview
========

.. rubric:: Class

A positive displacement compressor in which compression displacement is effected by the positive action of rotating elements (from `http://data.posccaesar.org/rdl/RDS435374 <http://data.posccaesar.org/rdl/RDS435374>`__).

.. image:: RotaryCompressor.*


.. rubric:: Supertypes

* :doc:`Compressor </reference/Equipment/Compressor>`

.. _dexpi-Equipment-RotaryCompressor-DesignRotationalSpeed:

DesignRotationalSpeed
=====================

.. rubric:: Attribute (data)


The rotational speed for which the :doc:`RotaryCompressor </reference/Equipment/RotaryCompressor>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullableRotationalFrequency </reference/PhysicalQuantities/NullableRotationalFrequency>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN ROTATIONAL SPEED <http://sandbox.dexpi.org/rdl/DesignRotationalSpeed>`__

   Name
       DesignRotationalSpeed

   AttributeURI
       `http://sandbox.dexpi.org/rdl/DesignRotationalSpeed <http://sandbox.dexpi.org/rdl/DesignRotationalSpeed>`__

.. admonition:: example

   The instance rotaryCompressor1 represents a :doc:`RotaryCompressor </reference/Equipment/RotaryCompressor>` with a :ref:`DesignRotationalSpeed <dexpi-Equipment-RotaryCompressor-DesignRotationalSpeed>` of 180.0 min\ :sup:`-1`.

   .. image:: /diagrams/diagram417.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="rotaryCompressor1"
            ComponentClass="RotaryCompressor"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS435374" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="DesignRotationalSpeed"
              AttributeURI="http://sandbox.dexpi.org/rdl/DesignRotationalSpeed"
              Format="double"
              Value="180.0"
              Units="ReciprocalMinute"
              UnitsURI="http://data.posccaesar.org/rdl/RDS4316851589" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
