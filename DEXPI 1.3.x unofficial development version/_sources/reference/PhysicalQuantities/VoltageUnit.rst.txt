===========
VoltageUnit
===========

Overview
========

.. rubric:: Enumeration

A unit of measurement for a physical quantity of type :doc:`NullableVoltage </reference/PhysicalQuantities/NullableVoltage>` with :ref:`dimension <dexpi-glossary-physical-dimension>` L\ :sup:`2`\MT\ :sup:`-3`\I\ :sup:`-1`.

.. admonition:: proteus

   :doc:`VoltageUnit </reference/PhysicalQuantities/VoltageUnit>` is only used as the type of the :ref:`Unit <dexpi-PhysicalQuantities-Voltage-Unit>` attribute of :doc:`Voltage </reference/PhysicalQuantities/Voltage>`. :doc:`Voltage </reference/PhysicalQuantities/Voltage>` is implemented using :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`. In a ``<GenericAttribute>`` element, the :doc:`VoltageUnit </reference/PhysicalQuantities/VoltageUnit>` literal is given by means of its :ref:`RDL reference <proteus-concept-rdl-reference>` in the table above. The ``Units`` attribute of the element is the name of the RDL reference in camel-case; it equals the name of the literal. The ``UnitsURI`` attribute of the element is the URI of the RDL reference.
