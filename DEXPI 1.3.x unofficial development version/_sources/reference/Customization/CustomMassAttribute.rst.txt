.. admonition:: example

   As :doc:`CustomAttributeOwner </reference/Customization/CustomAttributeOwner>` is abstract, we consider :doc:`Sieve </reference/Equipment/Sieve>` as an arbitrary concrete subclass.

   The :doc:`Sieve </reference/Equipment/Sieve>` sieve1 has a :doc:`CustomMassAttribute </reference/Customization/CustomMassAttribute>` with :ref:`AttributeName <dexpi-Customization-CustomAttribute-AttributeName>` "UpperLimitDesignLoad" and :ref:`AttributeURI <dexpi-Customization-CustomAttribute-AttributeURI>` "http://sandbox.dexpi.org/rdl/UpperLimitDesignLoad"DOTANDSPACEHACKThe :ref:`Value <dexpi-Customization-CustomMassAttribute-Value>` is 200 kg.

   .. image:: /diagrams/diagram21.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="sieve1"
            ComponentClass="Sieve"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/Sieve" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiCustomAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="UpperLimitDesignLoad"
              AttributeURI="http://sandbox.dexpi.org/rdl/UpperLimitDesignLoad"
              Format="double"
              Type="Mass"
              TypeURI="http://data.posccaesar.org/rdl/RDS353339"
              Value="200"
              Units="Kilogram"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1328669" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
