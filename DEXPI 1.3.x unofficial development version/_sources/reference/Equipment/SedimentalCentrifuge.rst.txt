====================
SedimentalCentrifuge
====================

Overview
========

.. rubric:: Class

A centrifuge that is intended to separate solids from liquids by a centrifugal process based on different densities.

.. image:: SedimentalCentrifuge.*


.. rubric:: Supertypes
