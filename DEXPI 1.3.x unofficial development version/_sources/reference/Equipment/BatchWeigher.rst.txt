DesignCapacityWeighingQuantities
================================

.. rubric:: Attribute (data)


The capacity for the number of weighing quantities per time for which the :doc:`BatchWeigher </reference/Equipment/BatchWeigher>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullableNumberPerTimeInterval </reference/PhysicalQuantities/NullableNumberPerTimeInterval>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN CAPACITY WEIGHING QUANTITIES <http://sandbox.dexpi.org/rdl/DesignCapacityWeighingQuantities>`__

   Name
       DesignCapacityWeighingQuantities

   AttributeURI
       `http://sandbox.dexpi.org/rdl/DesignCapacityWeighingQuantities <http://sandbox.dexpi.org/rdl/DesignCapacityWeighingQuantities>`__

.. admonition:: example

   The instance batchWeigher1 represents a :doc:`BatchWeigher </reference/Equipment/BatchWeigher>` with a :ref:`DesignCapacityWeighingQuantities <dexpi-Equipment-BatchWeigher-DesignCapacityWeighingQuantities>` of 42.0 s\ :sup:`-1`.

   .. image:: /diagrams/diagram115.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="batchWeigher1"
            ComponentClass="BatchWeigher"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/BatchWeigher" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">

              Name="DesignCapacityWeighingQuantities"
              AttributeURI="http://sandbox.dexpi.org/rdl/DesignCapacityWeighingQuantities"
