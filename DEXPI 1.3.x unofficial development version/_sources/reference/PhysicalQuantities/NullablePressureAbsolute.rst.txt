========================
NullablePressureAbsolute
========================

Overview
========

.. rubric:: Abstract data type

A physical quantity of dimension L\ :sup:`-1`\MT\ :sup:`-2` (inherited from :doc:`NullablePressure </reference/PhysicalQuantities/NullablePressure>`\ ) for the application type *absolute pressure*. A *pressure absolute* is a pressure relative to a perfect vacuum. This data type is also used for the *difference between two pressures* other than atmospheric pressure. :doc:`NullablePressureAbsolute </reference/PhysicalQuantities/NullablePressureAbsolute>` is abstract and has two concrete subtypes:

* a :doc:`PressureAbsolute </reference/PhysicalQuantities/PressureAbsolute>` is an :ref:`actual value <dexpi-concept-actual-value>` for a physical quantity with a numerical value and a unit of measurement;

* a :doc:`NullPressureAbsolute </reference/PhysicalQuantities/NullPressureAbsolute>` is a :ref:`null value <dexpi-concept-null-value>` that explicitly indicates the absence of an actual physical quantity.
