.. _dexpi-Equipment-AgitatorRotor-Chamber:

Chamber
=======

.. rubric:: Attribute (reference)


The :doc:`Chamber </reference/Equipment/Chamber>` in which the :doc:`AgitatorRotor </reference/Equipment/AgitatorRotor>` is located, if applicable. The Chamber must be a component of the same object as the :doc:`AgitatorRotor </reference/Equipment/AgitatorRotor>`.

Multiplicity
    0..1

Type
    :doc:`Chamber </reference/Equipment/Chamber>`

Opposite multiplicity
    0..*

.. admonition:: proteus

   The attribute is implemented using :ref:`Proteus \<Association\> elements <proteus-concept-reference-property-association>`.

   Association type for the attribute ower
       ``"is located in"``

   Opposite association type
       ``"is the location of"``

.. admonition:: example

   .. image:: /diagrams/diagram87.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="agitatorRotor1"
            ComponentClass="AgitatorRotor"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/AgitatorRotor" DOTS123="DOTS123">
          ...
          <Association
            Type="is located in"
            ItemID="chamber1" />
          ...
        <Equipment />
        ...
        <Equipment
            ID="chamber1"
            ComponentClass="Chamber"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS903151421" DOTS123="DOTS123">
          ...
          <Association
            Type="is the location of"
            ItemID="agitatorRotor1" />
          ...
        <Equipment />
