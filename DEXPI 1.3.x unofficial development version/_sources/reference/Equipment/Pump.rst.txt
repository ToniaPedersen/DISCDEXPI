====
Pump
====

Overview
========

.. rubric:: Class

A machine that is capable of pumping but may require parts and subsystems for that capability.

.. image:: Pump.*

.. _dexpi-Equipment-Pump-DesignPressureHead:

DesignPressureHead
==================

.. rubric:: Attribute (data)


The pressure head for which the :doc:`Pump </reference/Equipment/Pump>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullableLength </reference/PhysicalQuantities/NullableLength>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN PRESSURE HEAD <http://sandbox.dexpi.org/rdl/DesignPressureHead>`__

   Name
       DesignPressureHead

   AttributeURI
       `http://sandbox.dexpi.org/rdl/DesignPressureHead <http://sandbox.dexpi.org/rdl/DesignPressureHead>`__

.. admonition:: example

   The instance pump1 represents a :doc:`Pump </reference/Equipment/Pump>` with a :ref:`DesignPressureHead <dexpi-Equipment-Pump-DesignPressureHead>` of 40.0 m.

   .. image:: /diagrams/diagram393.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="pump1"
            ComponentClass="Pump"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS327239" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="DesignPressureHead"
              AttributeURI="http://sandbox.dexpi.org/rdl/DesignPressureHead"
              Format="double"
              Value="40.0"
              Units="Metre"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1332674" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
