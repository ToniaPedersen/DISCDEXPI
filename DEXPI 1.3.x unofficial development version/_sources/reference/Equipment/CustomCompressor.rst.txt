================
CustomCompressor
================

Overview
========

.. rubric:: Class

A :ref:`custom <dexpi-concept-custom-sibling>` :doc:`Compressor </reference/Equipment/Compressor>`, i.e., a :doc:`Compressor </reference/Equipment/Compressor>` that is not covered by any of the other subclasses of :doc:`Compressor </reference/Equipment/Compressor>` (:doc:`AirEjector </reference/Equipment/AirEjector>`, :doc:`AxialCompressor </reference/Equipment/AxialCompressor>`, :doc:`CentrifugalCompressor </reference/Equipment/CentrifugalCompressor>`, :doc:`ReciprocatingCompressor </reference/Equipment/ReciprocatingCompressor>`, or :doc:`RotaryCompressor </reference/Equipment/RotaryCompressor>`\ ).

.. image:: CustomCompressor.*


.. rubric:: Supertypes

* :doc:`Compressor </reference/Equipment/Compressor>`

* :doc:`CustomObject </reference/Customization/CustomObject>`

.. _dexpi-Equipment-CustomCompressor-Displacers:

Displacers
==========

.. rubric:: Attribute (composition)


The displacers of the :doc:`CustomCompressor </reference/Equipment/CustomCompressor>`.

Multiplicity
    \*

Type
    :doc:`Displacer </reference/Equipment/Displacer>`

Opposite multiplicity
    0..1

.. admonition:: proteus

   The attribute is implemented using the :ref:`XML hierarchy of the Proteus file <proteus-concept-composition-property-xml-child>`: The  element for the attribute value (a :doc:`Displacer </reference/Equipment/Displacer>`\ ) is a child of the ``<Equipment>`` element for the attribute owner (a :doc:`CustomCompressor </reference/Equipment/CustomCompressor>`\ ).

.. admonition:: example

   .. image:: /diagrams/diagram194.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="customCompressor1"
            ComponentClass="CustomCompressor"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/CustomCompressor" DOTS123="DOTS123">
          ...
          <Equipment
              ID="displacer1"
              ComponentClass="Displacer"
              ComponentClassURI="http://sandbox.dexpi.org/rdl/Displacer" DOTS123="DOTS123">
            ...
          <Equipment />
          ...
        <Equipment />
