===============
PackagingSystem
===============

Overview
========

.. rubric:: Class

A system that is intended for the preparation of goods for transport, warehousing, logistics, sale, and end use (from `http://data.15926.org/rdl/RDS2228725 <http://data.15926.org/rdl/RDS2228725>`__).

.. image:: PackagingSystem.*


.. rubric:: Supertypes
