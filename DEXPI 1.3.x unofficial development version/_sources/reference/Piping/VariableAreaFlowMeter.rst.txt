=====================
VariableAreaFlowMeter
=====================

Overview
========

.. rubric:: Class

A flow meter consisting of a vertical tube with a conically shaped bore which widens to the top in which a solid body (float) is supported by the force exerted by the fluid stream (from `http://data.posccaesar.org/rdl/RDS418229 <http://data.posccaesar.org/rdl/RDS418229>`__).

.. image:: VariableAreaFlowMeter.*


.. rubric:: Supertypes

* :doc:`InlinePrimaryElement </reference/Piping/InlinePrimaryElement>`

   The class is implemented as a :ref:`Proteus XML element with RDL reference <proteus-concept-xml-element-with-rdl>`.

   Tag

   RDL reference
       `VARIABLE AREA FLOW METER <http://data.posccaesar.org/rdl/RDS418229>`__

   ComponentClass

       `http://data.posccaesar.org/rdl/RDS418229 <http://data.posccaesar.org/rdl/RDS418229>`__
