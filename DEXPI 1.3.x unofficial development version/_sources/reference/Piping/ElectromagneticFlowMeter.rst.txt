========================
ElectromagneticFlowMeter
========================

Overview
========

.. rubric:: Class

A velocity flow meter that is measuring flow rate of a conductive fluid running through a magnetic field by measuring the charge created when fluid interacting with the field (from `http://data.posccaesar.org/rdl/RDS1009664 <http://data.posccaesar.org/rdl/RDS1009664>`__).

.. image:: ElectromagneticFlowMeter.*
