.. _dexpi-PlantStructure-TechnicalItem-ParentStructure:

ParentStructure
===============

.. rubric:: Attribute (reference)


A superordinate structure of which the :doc:`TechnicalItem </reference/PlantStructure/TechnicalItem>` is a part.

Multiplicity
    0..1

Type
    :doc:`TechnicalItemParentStructure </reference/PlantStructure/TechnicalItemParentStructure>`

Opposite multiplicity
    0..*

.. admonition:: proteus

   The attribute is implemented using :ref:`Proteus \<Association\> elements <proteus-concept-reference-property-association>`.

   Association type for the attribute ower
       ``"is a part of"``

   Opposite association type
       ``"is a collection including"``

.. admonition:: example

   .. image:: /diagrams/diagram753.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="tank1"
            ComponentClass="Tank"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS445139" DOTS123="DOTS123">
          ...
          <Association
            Type="is a part of"
            ItemID="processPlant1" />
          ...
        <Equipment />
        ...
        <PlantStructureItem
            ID="processPlant1"
            ComponentClass="ProcessPlant"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS7151859" DOTS123="DOTS123">
          ...
          <Association
            Type="is a collection including"
            ItemID="tank1" />
          ...
        <PlantStructureItem />
