===========================
NullableRotationalFrequency
===========================

Overview
========

.. rubric:: Abstract data type

A physical quantity of dimension T\ :sup:`-1` (inherited from :doc:`NullableFrequency </reference/PhysicalQuantities/NullableFrequency>`\ ) for the application type *rotational frequency*. :doc:`NullableRotationalFrequency </reference/PhysicalQuantities/NullableRotationalFrequency>` is abstract and has two concrete subtypes:

* a :doc:`RotationalFrequency </reference/PhysicalQuantities/RotationalFrequency>` is an :ref:`actual value <dexpi-concept-actual-value>` for a physical quantity with a numerical value and a unit of measurement;

* a :doc:`NullRotationalFrequency </reference/PhysicalQuantities/NullRotationalFrequency>` is a :ref:`null value <dexpi-concept-null-value>` that explicitly indicates the absence of an actual physical quantity.
