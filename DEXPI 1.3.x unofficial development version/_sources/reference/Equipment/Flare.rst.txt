=====
Flare
=====

Overview
========

.. rubric:: Class

An artefact and waste gas emitter that is intended to burn waste gas in secure distance from the plant or platform.

.. image:: Flare.*


.. rubric:: Supertypes

* :doc:`WasteGasEmitter </reference/Equipment/WasteGasEmitter>`

.. admonition:: proteus

   The class is implemented as a :ref:`Proteus XML element with RDL reference <proteus-concept-xml-element-with-rdl>`.

   Tag
       ``<Equipment>``

   RDL reference
       `FLARE <http://sandbox.dexpi.org/rdl/Flare>`__

   ComponentClass
       Flare

   ComponentClassURI
       `http://sandbox.dexpi.org/rdl/Flare <http://sandbox.dexpi.org/rdl/Flare>`__
