PressureGaugeUnit
=================

.. admonition:: proteus

   :doc:`PressureGaugeUnit </reference/PhysicalQuantities/PressureGaugeUnit>` is only used as the type of the :ref:`Unit <dexpi-PhysicalQuantities-PressureGauge-Unit>` attribute of :doc:`PressureGauge </reference/PhysicalQuantities/PressureGauge>`. :doc:`PressureGauge </reference/PhysicalQuantities/PressureGauge>` is implemented using :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`. In a ``<GenericAttribute>`` element, the :doc:`PressureGaugeUnit </reference/PhysicalQuantities/PressureGaugeUnit>` literal is given by means of its :ref:`RDL reference <proteus-concept-rdl-reference>` in the table above. The ``Units`` attribute of the element is the name of the RDL reference in camel-case; it equals the name of the literal. The ``UnitsURI`` attribute of the element is the URI of the RDL reference.
