===============
CentrifugalPump
===============

Overview
========

.. rubric:: Class

A dynamic pump utilizing impellers provided with vanes generating centrifugal force to achieve the required pressure head (from `http://data.posccaesar.org/rdl/RDS416834 <http://data.posccaesar.org/rdl/RDS416834>`__).

.. image:: CentrifugalPump.*


.. rubric:: Supertypes

* :doc:`Pump </reference/Equipment/Pump>`

DesignRotationalSpeed
=====================

.. rubric:: Attribute (data)


The rotational speed for which the :doc:`CentrifugalPump </reference/Equipment/CentrifugalPump>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullableRotationalFrequency </reference/PhysicalQuantities/NullableRotationalFrequency>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN ROTATIONAL SPEED <http://sandbox.dexpi.org/rdl/DesignRotationalSpeed>`__

   Name
       DesignRotationalSpeed

   AttributeURI
       `http://sandbox.dexpi.org/rdl/DesignRotationalSpeed <http://sandbox.dexpi.org/rdl/DesignRotationalSpeed>`__

.. admonition:: example

   The instance centrifugalPump1 represents a :doc:`CentrifugalPump </reference/Equipment/CentrifugalPump>` with a :ref:`DesignRotationalSpeed <dexpi-Equipment-CentrifugalPump-DesignRotationalSpeed>` of 180.0 min\ :sup:`-1`.

   .. image:: /diagrams/diagram134.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="centrifugalPump1"
            ComponentClass="CentrifugalPump"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS416834" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="DesignRotationalSpeed"
              AttributeURI="http://sandbox.dexpi.org/rdl/DesignRotationalSpeed"
              Format="double"
              Value="180.0"
              Units="ReciprocalMinute"
              UnitsURI="http://data.posccaesar.org/rdl/RDS4316851589" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
