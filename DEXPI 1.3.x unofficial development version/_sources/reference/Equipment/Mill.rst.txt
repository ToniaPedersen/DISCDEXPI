====
Mill
====

Overview
========

.. rubric:: Class

A physical object for grinding or pulverizing materials. Also a machine for shaping metal. In general a machine that manufactures by the continuous repetition of some simple action (from `http://data.posccaesar.org/rdl/RDS11589220 <http://data.posccaesar.org/rdl/RDS11589220>`__).

.. image:: Mill.*


.. rubric:: Supertypes

* :doc:`Equipment </reference/Equipment/Equipment>`

.. rubric:: Subtypes

* :doc:`Crusher </reference/Equipment/Crusher>`

* :doc:`CustomMill </reference/Equipment/CustomMill>`

* :doc:`Grinder </reference/Equipment/Grinder>`

   Name

   AttributeURI
       `http://sandbox.dexpi.org/rdl/DesignCapacityMassFlowRate <http://sandbox.dexpi.org/rdl/DesignCapacityMassFlowRate>`__
