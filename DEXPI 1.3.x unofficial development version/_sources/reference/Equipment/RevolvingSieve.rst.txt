DesignRotationalFrequency
=========================

.. rubric:: Attribute (data)


The rotational frequency for which the :doc:`RevolvingSieve </reference/Equipment/RevolvingSieve>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullableRotationalFrequency </reference/PhysicalQuantities/NullableRotationalFrequency>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN ROTATIONAL FREQUENCY <http://sandbox.dexpi.org/rdl/DesignRotationalFrequency>`__

   Name
       DesignRotationalFrequency

   AttributeURI
       `http://sandbox.dexpi.org/rdl/DesignRotationalFrequency <http://sandbox.dexpi.org/rdl/DesignRotationalFrequency>`__

   The instance revolvingSieve1 represents a :doc:`RevolvingSieve </reference/Equipment/RevolvingSieve>` with a :ref:`DesignRotationalFrequency <dexpi-Equipment-RevolvingSieve-DesignRotationalFrequency>` of 180.0 min\ :sup:`-1`.

   .. image:: /diagrams/diagram414.*


   .. admonition:: proteus-example

          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
