.. _dexpi-Equipment-SedimentalCentrifugeDrum-Chamber:

Chamber
=======

.. rubric:: Attribute (reference)


The :doc:`Chamber </reference/Equipment/Chamber>` in which the :doc:`SedimentalCentrifugeDrum </reference/Equipment/SedimentalCentrifugeDrum>` is located, if applicable. The Chamber must be a component of the same object as the :doc:`SedimentalCentrifugeDrum </reference/Equipment/SedimentalCentrifugeDrum>`.

Multiplicity
    0..1

Type
    :doc:`Chamber </reference/Equipment/Chamber>`

Opposite multiplicity
    0..*

.. admonition:: proteus

   The attribute is implemented using :ref:`Proteus \<Association\> elements <proteus-concept-reference-property-association>`.

   Association type for the attribute ower
       ``"is located in"``

   Opposite association type
       ``"is the location of"``

.. admonition:: example

   .. image:: /diagrams/diagram443.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="sedimentalCentrifugeDrum1"
            ComponentClass="SedimentalCentrifugeDrum"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/SedimentalCentrifugeDrum" DOTS123="DOTS123">
          ...
          <Association
            Type="is located in"
            ItemID="chamber1" />
          ...
        <Equipment />
        ...
        <Equipment
            ID="chamber1"

    0..1

Type
    :doc:`NullableLength </reference/PhysicalQuantities/NullableLength>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DIAMETER <http://data.posccaesar.org/rdl/RDS350954>`__

   Name
       Diameter

   AttributeURI
       `http://data.posccaesar.org/rdl/RDS350954 <http://data.posccaesar.org/rdl/RDS350954>`__

.. admonition:: example

   The instance sedimentalCentrifugeDrum1 represents a :doc:`SedimentalCentrifugeDrum </reference/Equipment/SedimentalCentrifugeDrum>` with a :ref:`Diameter <dexpi-Equipment-SedimentalCentrifugeDrum-Diameter>` of 20.0 cm.
