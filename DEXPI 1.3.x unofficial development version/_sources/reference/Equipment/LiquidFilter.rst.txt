DesignCapacityVolumeFlowRate
============================

.. rubric:: Attribute (data)


The volume flow rate for which the :doc:`LiquidFilter </reference/Equipment/LiquidFilter>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullableVolumeFlowRate </reference/PhysicalQuantities/NullableVolumeFlowRate>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN CAPACITY VOLUME FLOW RATE <http://sandbox.dexpi.org/rdl/DesignCapacityVolumeFlowRate>`__

   Name
       DesignCapacityVolumeFlowRate

   AttributeURI
       `http://sandbox.dexpi.org/rdl/DesignCapacityVolumeFlowRate <http://sandbox.dexpi.org/rdl/DesignCapacityVolumeFlowRate>`__

.. admonition:: example

   The instance liquidFilter1 represents a :doc:`LiquidFilter </reference/Equipment/LiquidFilter>` with a :ref:`DesignCapacityVolumeFlowRate <dexpi-Equipment-LiquidFilter-DesignCapacityVolumeFlowRate>` of 420.0 m\ :sup:`3`/h.

   .. image:: /diagrams/diagram343.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="liquidFilter1"
            ComponentClass="LiquidFilter"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/LiquidFilter" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="DesignCapacityVolumeFlowRate"
              AttributeURI="http://sandbox.dexpi.org/rdl/DesignCapacityVolumeFlowRate"
              Format="double"
              Value="420.0"
              Units="MetreCubedPerHour"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1321064" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
