============
CustomVessel
============

Overview
========

.. rubric:: Class

A :ref:`custom <dexpi-concept-custom-sibling>` :doc:`Vessel </reference/Equipment/Vessel>`, i.e., a :doc:`Vessel </reference/Equipment/Vessel>` that is not covered by any of the other subclasses of :doc:`Vessel </reference/Equipment/Vessel>` (:doc:`PressureVessel </reference/Equipment/PressureVessel>`, :doc:`Silo </reference/Equipment/Silo>`, or :doc:`Tank </reference/Equipment/Tank>`\ ).

.. image:: CustomVessel.*


.. rubric:: Supertypes

* :doc:`CustomObject </reference/Customization/CustomObject>`

* :doc:`Vessel </reference/Equipment/Vessel>`

| Name                                                                                               | Multiplicity | Type                                                                                               |

| :ref:`CylinderLength <dexpi-Equipment-CustomVessel-CylinderLength>`                                | 0..1         | :doc:`NullableLength </reference/PhysicalQuantities/NullableLength>`                               |
+----------------------------------------------------------------------------------------------------+--------------+----------------------------------------------------------------------------------------------------+

.. admonition:: proteus

   The class is implemented as a :ref:`Proteus XML element with RDL reference <proteus-concept-xml-element-with-rdl>`.
