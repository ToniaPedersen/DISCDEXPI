=========
PowerUnit
=========

Overview
========

.. rubric:: Enumeration

A unit of measurement for a physical quantity of type :doc:`NullablePower </reference/PhysicalQuantities/NullablePower>` with :ref:`dimension <dexpi-glossary-physical-dimension>` L\ :sup:`2`\MT\ :sup:`-3`.

.. admonition:: proteus

   :doc:`PowerUnit </reference/PhysicalQuantities/PowerUnit>` is only used as the type of the :ref:`Unit <dexpi-PhysicalQuantities-Power-Unit>` attribute of :doc:`Power </reference/PhysicalQuantities/Power>`. :doc:`Power </reference/PhysicalQuantities/Power>` is implemented using :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`. In a ``<GenericAttribute>`` element, the :doc:`PowerUnit </reference/PhysicalQuantities/PowerUnit>` literal is given by means of its :ref:`RDL reference <proteus-concept-rdl-reference>` in the table above. The ``Units`` attribute of the element is the name of the RDL reference in camel-case; it equals the name of the literal. The ``UnitsURI`` attribute of the element is the URI of the RDL reference.
