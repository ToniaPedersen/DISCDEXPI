======================
DirectCurrentGenerator
======================

Overview
========

.. rubric:: Class

An :doc:`ElectricGenerator </reference/Equipment/ElectricGenerator>` and current generator for the production of direct current (DC).

.. image:: DirectCurrentGenerator.*


.. rubric:: Supertypes

* :doc:`ElectricGenerator </reference/Equipment/ElectricGenerator>`

.. admonition:: proteus

   The class is implemented as a :ref:`Proteus XML element with RDL reference <proteus-concept-xml-element-with-rdl>`.

   Tag
       ``<Equipment>``

   RDL reference
       `DIRECT CURRENT GENERATOR <http://sandbox.dexpi.org/rdl/DirectCurrentGenerator>`__

   ComponentClass
       DirectCurrentGenerator

   ComponentClassURI
       `http://sandbox.dexpi.org/rdl/DirectCurrentGenerator <http://sandbox.dexpi.org/rdl/DirectCurrentGenerator>`__
