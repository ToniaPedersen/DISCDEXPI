ParentStructure
===============

.. rubric:: Attribute (reference)


A superordinate structure of which the :doc:`PlantSection </reference/PlantStructure/PlantSection>` is a part.

Multiplicity
    0..1

Type
    :doc:`PlantSectionParentStructure </reference/PlantStructure/PlantSectionParentStructure>`

Opposite multiplicity
    0..*

.. admonition:: proteus

   The attribute is implemented using :ref:`Proteus \<Association\> elements <proteus-concept-reference-property-association>`.

   Association type for the attribute ower
       ``"is a part of"``

   Opposite association type
       ``"is a collection including"``

.. admonition:: example

   .. image:: /diagrams/diagram738.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <PlantStructureItem
            ID="plantSection1"
            ComponentClass="PlantSectionIso102092012"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/PlantSectionIso102092012" DOTS123="DOTS123">
          ...
          <Association
            Type="is a part of"
            ItemID="processPlant1" />
          ...
        <PlantStructureItem />
        ...
        <PlantStructureItem
            ID="processPlant1"
            ComponentClass="ProcessPlant"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS7151859" DOTS123="DOTS123">
          ...
          <Association
            Type="is a collection including"
            ItemID="plantSection1" />
          ...
        <PlantStructureItem />
