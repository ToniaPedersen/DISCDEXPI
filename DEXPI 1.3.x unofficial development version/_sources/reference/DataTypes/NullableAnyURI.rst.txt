==============
NullableAnyURI
==============

Overview
========

.. rubric:: Abstract data type

A Uniform Resource Identifier (URI), or a null value. :doc:`NullableAnyURI </reference/DataTypes/NullableAnyURI>` is abstract and has two concrete subtypes:

* an :doc:`AnyURI </reference/DataTypes/AnyURI>` is an :ref:`actual value <dexpi-concept-actual-value>` for a Uniform Resource Identifier (URI);

* a :doc:`NullAnyURI </reference/DataTypes/NullAnyURI>` is a :ref:`null value <dexpi-concept-null-value>` that explicitly indicates the absence of a Uniform Resource Identifier (URI).
