===============================
NullableHeatTransferCoefficient
===============================

Overview
========

.. rubric:: Abstract data type

:doc:`NullableHeatTransferCoefficient </reference/PhysicalQuantities/NullableHeatTransferCoefficient>` is a :ref:`simple physical quantity type <dexpi-concept-simple-physical-quantity-type>` for the dimension MT\ :sup:`-3`\Î˜. :doc:`NullableHeatTransferCoefficient </reference/PhysicalQuantities/NullableHeatTransferCoefficient>` is abstract and has two concrete subtypes:

* a :doc:`HeatTransferCoefficient </reference/PhysicalQuantities/HeatTransferCoefficient>` is an :ref:`actual value <dexpi-concept-actual-value>` for a physical quantity with a numerical value and a unit of measurement;

* a :doc:`NullHeatTransferCoefficient </reference/PhysicalQuantities/NullHeatTransferCoefficient>` is a :ref:`null value <dexpi-concept-null-value>` that explicitly indicates the absence of an actual physical quantity.
