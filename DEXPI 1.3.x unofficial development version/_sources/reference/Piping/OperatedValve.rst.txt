=============
OperatedValve
=============

Overview
========

.. rubric:: Class

A valve that includes an external means of operation. (E.g. handwheel / lever / actuator.) (from `http://data.posccaesar.org/rdl/RDS11141590 <http://data.posccaesar.org/rdl/RDS11141590>`__).
