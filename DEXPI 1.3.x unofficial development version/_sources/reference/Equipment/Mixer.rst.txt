=====
Mixer
=====

Overview
========

.. rubric:: Class

An apparatus or machine that has the capability of mixing (from `http://data.15926.org/rdl/RDS222370 <http://data.15926.org/rdl/RDS222370>`__).

.. image:: Mixer.*

* :doc:`Equipment </reference/Equipment/Equipment>`

* :doc:`Kneader </reference/Equipment/Kneader>`

* :doc:`RotaryMixer </reference/Equipment/RotaryMixer>`

* :doc:`StaticMixer </reference/Equipment/StaticMixer>`

.. _dexpi-Equipment-Mixer-MixingElementAssemblies:

MixingElementAssemblies
=======================

.. rubric:: Attribute (composition)


The mixing element assemblies of the :doc:`Mixer </reference/Equipment/Mixer>`, if applicable.

Multiplicity
    \*

Type
    :doc:`MixingElementAssembly </reference/Equipment/MixingElementAssembly>`

Opposite multiplicity
    0..1

.. admonition:: proteus

   The attribute is implemented using the :ref:`XML hierarchy of the Proteus file <proteus-concept-composition-property-xml-child>`: The  element for the attribute value (a :doc:`MixingElementAssembly </reference/Equipment/MixingElementAssembly>`\ ) is a child of the ``<Equipment>`` element for the attribute owner (a :doc:`Mixer </reference/Equipment/Mixer>`\ ).

.. admonition:: example

   .. image:: /diagrams/diagram362.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="mixer1"
            ComponentClass="Mixer"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/Mixer" DOTS123="DOTS123">
          ...
          <Equipment
              ID="mixingElementAssembly1"
              ComponentClass="MixingElementAssembly"
              ComponentClassURI="http://sandbox.dexpi.org/rdl/MixingElementAssembly" DOTS123="DOTS123">
            ...
          <Equipment />
          ...
        <Equipment />
