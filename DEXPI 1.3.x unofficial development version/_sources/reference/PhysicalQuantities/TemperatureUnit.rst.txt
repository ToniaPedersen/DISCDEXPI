===============
TemperatureUnit
===============

Overview
========

.. rubric:: Enumeration

A unit of measurement for a physical quantity of type :doc:`NullableTemperature </reference/PhysicalQuantities/NullableTemperature>` with :ref:`dimension <dexpi-glossary-physical-dimension>` Θ.

.. admonition:: proteus

   :doc:`TemperatureUnit </reference/PhysicalQuantities/TemperatureUnit>` is only used as the type of the :ref:`Unit <dexpi-PhysicalQuantities-Temperature-Unit>` attribute of :doc:`Temperature </reference/PhysicalQuantities/Temperature>`. :doc:`Temperature </reference/PhysicalQuantities/Temperature>` is implemented using :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`. In a ``<GenericAttribute>`` element, the :doc:`TemperatureUnit </reference/PhysicalQuantities/TemperatureUnit>` literal is given by means of its :ref:`RDL reference <proteus-concept-rdl-reference>` in the table above. The ``Units`` attribute of the element is the name of the RDL reference in camel-case; it equals the name of the literal. The ``UnitsURI`` attribute of the element is the URI of the RDL reference.
