==========
CustomMill
==========

Overview
========

.. rubric:: Class

A :ref:`custom <dexpi-concept-custom-sibling>` :doc:`Mill </reference/Equipment/Mill>`, i.e., a :doc:`Mill </reference/Equipment/Mill>` that is not covered by any of the other subclasses of :doc:`Mill </reference/Equipment/Mill>` (:doc:`Crusher </reference/Equipment/Crusher>` or :doc:`Grinder </reference/Equipment/Grinder>`\ ).

.. image:: CustomMill.*


.. rubric:: Supertypes

* :doc:`CustomObject </reference/Customization/CustomObject>`

* :doc:`Mill </reference/Equipment/Mill>`

.. rubric:: Attributes (composition)

+----------------------------------------------------------------------------------------------------+--------------+--------------------------------------------------------------------------------------------+
| Name                                                                                               | Multiplicity | Type                                                                                       |
+====================================================================================================+==============+============================================================================================+
| :ref:`CrusherElements <dexpi-Equipment-CustomMill-CrusherElements>`                                | \*           | :doc:`CrusherElement </reference/Equipment/CrusherElement>`                                |
+----------------------------------------------------------------------------------------------------+--------------+--------------------------------------------------------------------------------------------+
| :ref:`GrindingElements <dexpi-Equipment-CustomMill-GrindingElements>`                              | \*           | :doc:`GrindingElement </reference/Equipment/GrindingElement>`                              |
+----------------------------------------------------------------------------------------------------+--------------+--------------------------------------------------------------------------------------------+

.. admonition:: proteus

   The class is implemented as a :ref:`Proteus XML element with RDL reference <proteus-concept-xml-element-with-rdl>`.

   Tag
       ``<Equipment>``

   RDL reference
       `CUSTOM MILL <http://sandbox.dexpi.org/rdl/CustomMill>`__

   ComponentClass
       CustomMill

   ComponentClassURI
       `http://sandbox.dexpi.org/rdl/CustomMill <http://sandbox.dexpi.org/rdl/CustomMill>`__

.. admonition:: example

   .. image:: /diagrams/diagram212.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="customMill1"
            ComponentClass="CustomMill"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/CustomMill" DOTS123="DOTS123">
          ...
        </Equipment>
      


.. _dexpi-Equipment-CustomMill-CrusherElements:

CrusherElements
===============

.. rubric:: Attribute (composition)


The crusher elements of the :doc:`CustomMill </reference/Equipment/CustomMill>`.

Multiplicity
    \*

Type
    :doc:`CrusherElement </reference/Equipment/CrusherElement>`

Opposite multiplicity
    0..1

.. admonition:: proteus

   The attribute is implemented using the :ref:`XML hierarchy of the Proteus file <proteus-concept-composition-property-xml-child>`: The  element for the attribute value (a :doc:`CrusherElement </reference/Equipment/CrusherElement>`\ ) is a child of the ``<Equipment>`` element for the attribute owner (a :doc:`CustomMill </reference/Equipment/CustomMill>`\ ).

.. admonition:: example

   .. image:: /diagrams/diagram213.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="customMill1"
            ComponentClass="CustomMill"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/CustomMill" DOTS123="DOTS123">
          ...
          <Equipment
              ID="crusherElement1"
              ComponentClass="CrusherUnit"
              ComponentClassURI="http://sandbox.dexpi.org/rdl/CrusherUnit" DOTS123="DOTS123">
            ...
          <Equipment />
          ...
        <Equipment />
