=======
Integer
=======

Overview
========

.. rubric:: Data type

An integer number. The value space of :doc:`Integer </reference/DataTypes/Integer>` is the same as that of the XML Schema data type `integer <https://www.w3.org/TR/xmlschema-2/#integer>`__ as specified by the W3C Recommendation `XML Schema Part 2: Datatypes Second Edition <https://www.w3.org/TR/xmlschema-2>`__ from October 28, 2004.

.. image:: Integer.*


.. rubric:: Supertypes

* :doc:`NullableInteger </reference/DataTypes/NullableInteger>`

.. admonition:: proteus

   An :doc:`Integer </reference/DataTypes/Integer>` is implemented as a literal for the XML Schema data type `integer <https://www.w3.org/TR/xmlschema-2/#integer>`__.

   The way this literal is used in a Proteus XML document depends on the Proteus Schema implementations of the data attributes with type :doc:`NullableInteger </reference/DataTypes/NullableInteger>`. There is no attribute with direct type :doc:`Integer </reference/DataTypes/Integer>`.

   * :doc:`NullableInteger </reference/DataTypes/NullableInteger>` is the type of the :ref:`Value <dexpi-Customization-CustomIntegerAttribute-Value>` attribute of :doc:`CustomIntegerAttribute </reference/Customization/CustomIntegerAttribute>`. :doc:`CustomIntegerAttribute </reference/Customization/CustomIntegerAttribute>` is implemented as a :ref:`DEXPI custom generic attribute <proteus-concept-dexpi-custom-generic-attribute>`.

   * All other attributes with type :doc:`NullableInteger </reference/DataTypes/NullableInteger>` are implemented as :ref:`DEXPI generic attributes <proteus-concept-dexpi-generic-attribute>`. For example, see the :ref:`NumberOfPackings <dexpi-Equipment-ColumnPackingsArrangement-NumberOfPackings>` attribute of :doc:`ColumnPackingsArrangement </reference/Equipment/ColumnPackingsArrangement>`.
