====================
PipeOffPageConnector
====================

Overview
========

.. rubric:: Abstract class

A connector that indicates that a piping network segment is continued elsewhere, either on the same PID or on another PID. Graphically, it is usually represented as an arrow.

.. image:: PipeOffPageConnector.*

.. _dexpi-Piping-PipeOffPageConnector-ConnectorReference:

ConnectorReference
==================

.. rubric:: Attribute (composition)


A reference indicating to which other :doc:`PipeOffPageConnector </reference/Piping/PipeOffPageConnector>` this :doc:`PipeOffPageConnector </reference/Piping/PipeOffPageConnector>` is connected.

Multiplicity
    0..1

Type
    :doc:`PipeOffPageConnectorReference </reference/Piping/PipeOffPageConnectorReference>`

Opposite multiplicity
    0..1

.. admonition:: proteus

   The attribute is implemented using the :ref:`XML hierarchy of the Proteus file <proteus-concept-composition-property-xml-child>`: The  element for the attribute value (a :doc:`PipeOffPageConnectorReference </reference/Piping/PipeOffPageConnectorReference>`\ ) is a child of the ``<PipeOffPageConnector>`` element for the attribute owner (a :doc:`PipeOffPageConnector </reference/Piping/PipeOffPageConnector>`\ ).

.. admonition:: example

   As the owner type :doc:`PipeOffPageConnector </reference/Piping/PipeOffPageConnector>` is abstract, we consider :doc:`FlowInPipeOffPageConnector </reference/Piping/FlowInPipeOffPageConnector>` as an arbitrary concrete subclass. As the value type :doc:`PipeOffPageConnectorReference </reference/Piping/PipeOffPageConnectorReference>` is abstract, we consider :doc:`PipeOffPageConnectorObjectReference </reference/Piping/PipeOffPageConnectorObjectReference>` as an arbitrary concrete subclass.

   .. image:: /diagrams/diagram679.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml

            ID="flowInPipeOffPageConnector1"
            ComponentClass="FlowInPipeOffPageConnector"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/FlowInPipeOffPageConnector" DOTS123="DOTS123">
          ...

              ID="pipeOffPageConnectorObjectReference1"
              ComponentClass="PipeOffPageConnectorObjectReference"
              ComponentClassURI="http://sandbox.dexpi.org/rdl/PipeOffPageConnectorObjectReference" DOTS123="DOTS123">
            ...

          ...
        <PipeOffPageConnector />
