===============
NullableVoltage
===============

Overview
========

.. rubric:: Abstract data type

:doc:`NullableVoltage </reference/PhysicalQuantities/NullableVoltage>` is a :ref:`simple physical quantity type <dexpi-concept-simple-physical-quantity-type>` for the dimension L\ :sup:`2`\MT\ :sup:`-3`\I\ :sup:`-1`. :doc:`NullableVoltage </reference/PhysicalQuantities/NullableVoltage>` is abstract and has two concrete subtypes:

* a :doc:`Voltage </reference/PhysicalQuantities/Voltage>` is an :ref:`actual value <dexpi-concept-actual-value>` for a physical quantity with a numerical value and a unit of measurement;

* a :doc:`NullVoltage </reference/PhysicalQuantities/NullVoltage>` is a :ref:`null value <dexpi-concept-null-value>` that explicitly indicates the absence of an actual physical quantity.
