===========================
ProcessPlantParentStructure
===========================

Overview
========

.. rubric:: Abstract class

A :doc:`PlantStructureItem </reference/PlantStructure/PlantStructureItem>` that is a suitable :ref:`ParentStructure <dexpi-PlantStructure-ProcessPlant-ParentStructure>` of a :doc:`ProcessPlant </reference/PlantStructure/ProcessPlant>`.

.. image:: ProcessPlantParentStructure.*


.. rubric:: Subtypes

* :doc:`Enterprise </reference/PlantStructure/Enterprise>`

* :doc:`IndustrialComplex </reference/PlantStructure/IndustrialComplex>`

* :doc:`Site </reference/PlantStructure/Site>`

.. admonition:: proteus

   Implementation is subclass-specific.

.. admonition:: example

   As :doc:`ProcessPlantParentStructure </reference/PlantStructure/ProcessPlantParentStructure>` is abstract, we consider :doc:`Enterprise </reference/PlantStructure/Enterprise>` as an arbitrary concrete subclass.

   .. image:: /diagrams/diagram749.*
