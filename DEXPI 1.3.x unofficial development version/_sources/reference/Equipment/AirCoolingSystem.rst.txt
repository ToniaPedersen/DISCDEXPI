================
AirCoolingSystem
================

Overview
========

.. rubric:: Class

A cooling system which uses air as the cooling medium (from `http://data.posccaesar.org/rdl/RDS277379 <http://data.posccaesar.org/rdl/RDS277379>`__).

.. image:: AirCoolingSystem.*


.. rubric:: Supertypes

DesignPower
===========

.. rubric:: Attribute (data)


The power for which the :doc:`AirCoolingSystem </reference/Equipment/AirCoolingSystem>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullablePower </reference/PhysicalQuantities/NullablePower>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN POWER <http://sandbox.dexpi.org/rdl/DesignPower>`__

   Name
       DesignPower

   AttributeURI
       `http://sandbox.dexpi.org/rdl/DesignPower <http://sandbox.dexpi.org/rdl/DesignPower>`__

.. admonition:: example

   The instance airCoolingSystem1 represents an :doc:`AirCoolingSystem </reference/Equipment/AirCoolingSystem>` with a :ref:`DesignPower <dexpi-Equipment-AirCoolingSystem-DesignPower>` of 500.0 kW.

   .. image:: /diagrams/diagram91.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="airCoolingSystem1"
            ComponentClass="AirCoolingSystem"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS277379" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="DesignPower"
              AttributeURI="http://sandbox.dexpi.org/rdl/DesignPower"
              Format="double"
              Value="500.0"
              Units="Kilowatt"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1330919" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
