=================
NullableFrequency
=================

Overview
========

.. rubric:: Abstract data type

:doc:`NullableFrequency </reference/PhysicalQuantities/NullableFrequency>` is an :ref:`application-dependent physical quantity type <dexpi-concept-application-dependent-physical-quantity-type>` for the dimension T\ :sup:`-1`. It has 3 subtypes for different application areas.

.. image:: NullableFrequency.*


.. rubric:: Subtypes

* :doc:`NullableElectricalFrequency </reference/PhysicalQuantities/NullableElectricalFrequency>`

* :doc:`NullableNumberPerTimeInterval </reference/PhysicalQuantities/NullableNumberPerTimeInterval>`

* :doc:`NullableRotationalFrequency </reference/PhysicalQuantities/NullableRotationalFrequency>`

.. admonition:: proteus

   All data attributes with a type derived from :doc:`NullableFrequency </reference/PhysicalQuantities/NullableFrequency>` are implemented as :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

.. admonition:: example

   See the examples for :doc:`ElectricalFrequency </reference/PhysicalQuantities/ElectricalFrequency>`, :doc:`NullElectricalFrequency </reference/PhysicalQuantities/NullElectricalFrequency>`, :doc:`NumberPerTimeInterval </reference/PhysicalQuantities/NumberPerTimeInterval>`, :doc:`NullNumberPerTimeInterval </reference/PhysicalQuantities/NullNumberPerTimeInterval>`, :doc:`RotationalFrequency </reference/PhysicalQuantities/RotationalFrequency>`, and :doc:`NullRotationalFrequency </reference/PhysicalQuantities/NullRotationalFrequency>`.
