.. admonition:: proteus

   :doc:`NumberPerTimeIntervalUnit </reference/PhysicalQuantities/NumberPerTimeIntervalUnit>` is only used as the type of the :ref:`Unit <dexpi-PhysicalQuantities-NumberPerTimeInterval-Unit>` attribute of :doc:`NumberPerTimeInterval </reference/PhysicalQuantities/NumberPerTimeInterval>`. :doc:`NumberPerTimeInterval </reference/PhysicalQuantities/NumberPerTimeInterval>` is implemented using :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`. In a ``<GenericAttribute>`` element, the :doc:`NumberPerTimeIntervalUnit </reference/PhysicalQuantities/NumberPerTimeIntervalUnit>` literal is given by means of its :ref:`RDL reference <proteus-concept-rdl-reference>` in the table above. The ``Units`` attribute of the element is the name of the RDL reference in camel-case; it equals the name of the literal. The ``UnitsURI`` attribute of the element is the URI of the RDL reference.
