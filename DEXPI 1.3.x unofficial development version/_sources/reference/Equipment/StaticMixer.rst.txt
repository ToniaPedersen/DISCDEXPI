===========
StaticMixer
===========

Overview
========

.. rubric:: Class

A physical object that is intended to mix fluid by means of diverging the flow with static obstacles or by increasing locally the velocity.

.. image:: StaticMixer.*


.. rubric:: Supertypes

* :doc:`Mixer </reference/Equipment/Mixer>`

.. _dexpi-Equipment-StaticMixer-UpperLimitAllowableDesignPressureDrop:

UpperLimitAllowableDesignPressureDrop
=====================================

.. rubric:: Attribute (data)


The upper limit for the pressure drop for which the :doc:`StaticMixer </reference/Equipment/StaticMixer>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullablePressureAbsolute </reference/PhysicalQuantities/NullablePressureAbsolute>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `UPPER LIMIT ALLOWABLE DESIGN PRESSURE DROP <http://sandbox.dexpi.org/rdl/UpperLimitAllowableDesignPressureDrop>`__

   Name
       UpperLimitAllowableDesignPressureDrop

   AttributeURI
       `http://sandbox.dexpi.org/rdl/UpperLimitAllowableDesignPressureDrop <http://sandbox.dexpi.org/rdl/UpperLimitAllowableDesignPressureDrop>`__

.. admonition:: example

   The instance staticMixer1 represents a :doc:`StaticMixer </reference/Equipment/StaticMixer>` with an :ref:`UpperLimitAllowableDesignPressureDrop <dexpi-Equipment-StaticMixer-UpperLimitAllowableDesignPressureDrop>` of 2.0 bar.

   .. image:: /diagrams/diagram466.*
