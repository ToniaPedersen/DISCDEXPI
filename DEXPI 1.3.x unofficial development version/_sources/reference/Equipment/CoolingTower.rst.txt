============
CoolingTower
============

Overview
========

.. rubric:: Class

A cooler and an air cooled heat exchanger that is a tall structure through which air circulates by convection (from `http://data.posccaesar.org/rdl/RDS14072341 <http://data.posccaesar.org/rdl/RDS14072341>`__).

.. image:: CoolingTower.*


.. rubric:: Supertypes

* :doc:`Equipment </reference/Equipment/Equipment>`

* :doc:`CustomCoolingTower </reference/Equipment/CustomCoolingTower>`

* :doc:`DryCoolingTower </reference/Equipment/DryCoolingTower>`

* :doc:`SprayCooler </reference/Equipment/SprayCooler>`

* :doc:`WetCoolingTower </reference/Equipment/WetCoolingTower>`

DesignHeatFlowRate
==================

.. rubric:: Attribute (data)


The heat flow rate for which the :doc:`CoolingTower </reference/Equipment/CoolingTower>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullablePower </reference/PhysicalQuantities/NullablePower>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN HEAT FLOW RATE <http://sandbox.dexpi.org/rdl/DesignHeatFlowRate>`__

   Name
       DesignHeatFlowRate

   AttributeURI
       `http://sandbox.dexpi.org/rdl/DesignHeatFlowRate <http://sandbox.dexpi.org/rdl/DesignHeatFlowRate>`__

.. admonition:: example

   The instance coolingTower1 represents a :doc:`CoolingTower </reference/Equipment/CoolingTower>` with a :ref:`DesignHeatFlowRate <dexpi-Equipment-CoolingTower-DesignHeatFlowRate>` of 313.0 kW.

   .. image:: /diagrams/diagram176.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="coolingTower1"
            ComponentClass="CoolingTower"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS14072341" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="DesignHeatFlowRate"
              AttributeURI="http://sandbox.dexpi.org/rdl/DesignHeatFlowRate"
              Format="double"
              Value="313.0"
              Units="Kilowatt"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1330919" />
            ...
          </GenericAttributes>
