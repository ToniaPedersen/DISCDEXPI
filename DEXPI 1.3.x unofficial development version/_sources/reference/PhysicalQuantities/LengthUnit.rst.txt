.. admonition:: proteus

   :doc:`LengthUnit </reference/PhysicalQuantities/LengthUnit>` is only used as the type of the :ref:`Unit <dexpi-PhysicalQuantities-Length-Unit>` attribute of :doc:`Length </reference/PhysicalQuantities/Length>`. :doc:`Length </reference/PhysicalQuantities/Length>` is implemented using :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`. In a ``<GenericAttribute>`` element, the :doc:`LengthUnit </reference/PhysicalQuantities/LengthUnit>` literal is given by means of its :ref:`RDL reference <proteus-concept-rdl-reference>` in the table above. The ``Units`` attribute of the element is the name of the RDL reference in camel-case; it equals the name of the literal. The ``UnitsURI`` attribute of the element is the URI of the RDL reference.
