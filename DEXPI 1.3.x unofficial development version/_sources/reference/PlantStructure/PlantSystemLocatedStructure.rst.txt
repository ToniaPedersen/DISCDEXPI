.. _dexpi-PlantStructure-PlantSystemLocatedStructure-PlantSystem:

PlantSystem
===========

.. rubric:: Attribute (reference)


The :doc:`PlantSystem </reference/PlantStructure/PlantSystem>` in which the :doc:`PlantSystemLocatedStructure </reference/PlantStructure/PlantSystemLocatedStructure>` is located.

Multiplicity
    0..1

Type
    :doc:`PlantSystem </reference/PlantStructure/PlantSystem>`

Opposite multiplicity
    0..*

.. admonition:: proteus

   The attribute is implemented using :ref:`Proteus \<Association\> elements <proteus-concept-reference-property-association>`.

   Association type for the attribute ower
       ``"is located in"``

   Opposite association type
       ``"is the location of"``

.. admonition:: example

   .. image:: /diagrams/diagram743.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="tank1"
            ComponentClass="Tank"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS445139" DOTS123="DOTS123">
          ...
          <Association
            Type="is located in"
            ItemID="plantSystem1" />
          ...
        <Equipment />
        ...
        <PlantStructureItem
            ID="plantSystem1"
            ComponentClass="PlantSystem"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/PlantSystem" DOTS123="DOTS123">
          ...
          <Association
            Type="is the location of"
            ItemID="tank1" />
          ...
        <PlantStructureItem />
