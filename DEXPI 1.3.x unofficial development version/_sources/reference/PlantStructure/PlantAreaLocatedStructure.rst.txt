.. _dexpi-PlantStructure-PlantAreaLocatedStructure-PlantArea:

PlantArea
=========

.. rubric:: Attribute (reference)


The :doc:`PlantArea </reference/PlantStructure/PlantArea>` in which the :doc:`PlantAreaLocatedStructure </reference/PlantStructure/PlantAreaLocatedStructure>` is located.

Multiplicity
    0..1

Type
    :doc:`PlantArea </reference/PlantStructure/PlantArea>`

Opposite multiplicity
    0..*

.. admonition:: proteus

   The attribute is implemented using :ref:`Proteus \<Association\> elements <proteus-concept-reference-property-association>`.

   Association type for the attribute ower
       ``"is located in"``

   Opposite association type
       ``"is the location of"``

.. admonition:: example

   .. image:: /diagrams/diagram736.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="tank1"
            ComponentClass="Tank"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS445139" DOTS123="DOTS123">
          ...
          <Association
            Type="is located in"
            ItemID="plantArea1" />
          ...
        <Equipment />
        ...
        <PlantStructureItem
            ID="plantArea1"
            ComponentClass="AreaIsa95"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS10418236534" DOTS123="DOTS123">
          ...
          <Association
            Type="is the location of"
            ItemID="tank1" />
          ...
        <PlantStructureItem />
