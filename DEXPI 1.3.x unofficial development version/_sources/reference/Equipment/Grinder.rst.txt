.. _dexpi-Equipment-Grinder-GrindingElements:

GrindingElements
================

.. rubric:: Attribute (composition)


The grinding elements of the :doc:`Grinder </reference/Equipment/Grinder>`.

Multiplicity
    \*

Type
    :doc:`GrindingElement </reference/Equipment/GrindingElement>`

Opposite multiplicity
    0..1

.. admonition:: proteus

   The attribute is implemented using the :ref:`XML hierarchy of the Proteus file <proteus-concept-composition-property-xml-child>`: The  element for the attribute value (a :doc:`GrindingElement </reference/Equipment/GrindingElement>`\ ) is a child of the ``<Equipment>`` element for the attribute owner (a :doc:`Grinder </reference/Equipment/Grinder>`\ ).

.. admonition:: example

   .. image:: /diagrams/diagram314.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="grinder1"
            ComponentClass="Grinder"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/Grinder" DOTS123="DOTS123">
          ...
          <Equipment
              ID="grindingElement1"
              ComponentClass="GrindingElement"
              ComponentClassURI="http://sandbox.dexpi.org/rdl/GrindingElement" DOTS123="DOTS123">
            ...
          <Equipment />
          ...
        <Equipment />
