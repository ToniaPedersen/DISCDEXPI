.. admonition:: example

   As :doc:`CustomAttributeOwner </reference/Customization/CustomAttributeOwner>` is abstract, we consider :doc:`Positioner </reference/Instrumentation/Positioner>` as an arbitrary concrete subclass.

   The :doc:`Positioner </reference/Instrumentation/Positioner>` positioner1 has a :doc:`CustomElectricalFrequencyAttribute </reference/Customization/CustomElectricalFrequencyAttribute>` with :ref:`AttributeName <dexpi-Customization-CustomAttribute-AttributeName>` "AlternatingCurrentFrequency" and an (arbitrary) :ref:`AttributeURI <dexpi-Customization-CustomAttribute-AttributeURI>` "http://www.example.org/9c2c76fc470069"DOTANDSPACEHACKThe :ref:`Value <dexpi-Customization-CustomElectricalFrequencyAttribute-Value>` is 50 Hz.

   .. image:: /diagrams/diagram17.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <ActuatingSystemComponent
            ID="positioner1"
            ComponentClass="Positioner"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/Positioner" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiCustomAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="AlternatingCurrentFrequency"
              AttributeURI="http://www.example.org/9c2c76fc470069"
              Format="double"
              Type="ElectricalFrequency"
              TypeURI="http://data.posccaesar.org/rdl/RDS401399"
              Value="50"
              Units="Hertz"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1326464" />
            ...
          </GenericAttributes>
          ...
        </ActuatingSystemComponent>
