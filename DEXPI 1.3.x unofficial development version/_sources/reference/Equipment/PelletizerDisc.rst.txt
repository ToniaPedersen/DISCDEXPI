.. _dexpi-Equipment-PelletizerDisc-Diameter:

Diameter
========

.. rubric:: Attribute (data)


The diameter of the :doc:`PelletizerDisc </reference/Equipment/PelletizerDisc>`.

Multiplicity
    0..1

Type
    :doc:`NullableLength </reference/PhysicalQuantities/NullableLength>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DIAMETER <http://data.posccaesar.org/rdl/RDS350954>`__

   Name
       Diameter

   AttributeURI
       `http://data.posccaesar.org/rdl/RDS350954 <http://data.posccaesar.org/rdl/RDS350954>`__

.. admonition:: example

   The instance pelletizerDisc1 represents a :doc:`PelletizerDisc </reference/Equipment/PelletizerDisc>` with a :ref:`Diameter <dexpi-Equipment-PelletizerDisc-Diameter>` of 20.0 cm.

   .. image:: /diagrams/diagram383.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="pelletizerDisc1"
            ComponentClass="PelletingDisc"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/PelletingDisc" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="Diameter"
              AttributeURI="http://data.posccaesar.org/rdl/RDS350954"
              Format="double"
              Value="20.0"
              Units="Centimetre"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1318004" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
