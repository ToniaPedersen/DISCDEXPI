.. rubric:: Attribute (data)


The upper limit for the temperature for which the :doc:`Chamber </reference/Equipment/Chamber>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullableTemperature </reference/PhysicalQuantities/NullableTemperature>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `UPPER LIMIT DESIGN TEMPERATURE <http://data.posccaesar.org/rdl/RDS360449>`__

   Name
       UpperLimitDesignTemperature

   AttributeURI
       `http://data.posccaesar.org/rdl/RDS360449 <http://data.posccaesar.org/rdl/RDS360449>`__

.. admonition:: example

   The instance chamber1 represents a :doc:`Chamber </reference/Equipment/Chamber>` with an :ref:`UpperLimitDesignTemperature <dexpi-Equipment-Chamber-UpperLimitDesignTemperature>` of 100.0 °C.

   .. image:: /diagrams/diagram149.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="chamber1"
            ComponentClass="Chamber"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS903151421" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="UpperLimitDesignTemperature"
              AttributeURI="http://data.posccaesar.org/rdl/RDS360449"
              Format="double"
              Value="100.0"
              Units="DegreeCelsius"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1322684" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
