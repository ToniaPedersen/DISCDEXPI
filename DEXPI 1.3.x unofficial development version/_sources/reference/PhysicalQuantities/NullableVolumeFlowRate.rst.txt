======================
NullableVolumeFlowRate
======================

Overview
========

.. rubric:: Abstract data type

:doc:`NullableVolumeFlowRate </reference/PhysicalQuantities/NullableVolumeFlowRate>` is a :ref:`simple physical quantity type <dexpi-concept-simple-physical-quantity-type>` for the dimension L\ :sup:`3`\T\ :sup:`-1`. :doc:`NullableVolumeFlowRate </reference/PhysicalQuantities/NullableVolumeFlowRate>` is abstract and has two concrete subtypes:

* a :doc:`VolumeFlowRate </reference/PhysicalQuantities/VolumeFlowRate>` is an :ref:`actual value <dexpi-concept-actual-value>` for a physical quantity with a numerical value and a unit of measurement;

* a :doc:`NullVolumeFlowRate </reference/PhysicalQuantities/NullVolumeFlowRate>` is a :ref:`null value <dexpi-concept-null-value>` that explicitly indicates the absence of an actual physical quantity.
