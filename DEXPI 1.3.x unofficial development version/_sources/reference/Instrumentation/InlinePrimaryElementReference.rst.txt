InlinePrimaryElement
====================

.. rubric:: Attribute (reference)


The :doc:`InlinePrimaryElement </reference/Piping/InlinePrimaryElement>` referenced by the :doc:`InlinePrimaryElementReference </reference/Instrumentation/InlinePrimaryElementReference>`.

Multiplicity
    0..1

Type
    :doc:`InlinePrimaryElement </reference/Piping/InlinePrimaryElement>`

Opposite multiplicity
    0..1

.. admonition:: proteus

   The attribute is implemented using :ref:`Proteus \<Association\> elements <proteus-concept-reference-property-association>`.

   Association type for the attribute ower
       ``"refers to"``

   Opposite association type
       ``"is referenced by"``
