===================
ElectricalSeparator
===================

Overview
========

.. rubric:: Class

A separator that uses electromagnetic, magnetic or electrostatic forces to separate phases.

.. image:: ElectricalSeparator.*


.. rubric:: Supertypes

* :doc:`Separator </reference/Equipment/Separator>`

.. rubric:: Attributes (data)

+----------------------------------------------------------------------------------------------------+--------------+------------------------------------------------------------------------------------------------+
| Name                                                                                               | Multiplicity | Type                                                                                           |
+====================================================================================================+==============+================================================================================================+
| :ref:`DesignPower <dexpi-Equipment-ElectricalSeparator-DesignPower>`                               | 0..1         | :doc:`NullablePower </reference/PhysicalQuantities/NullablePower>`                             |
+----------------------------------------------------------------------------------------------------+--------------+------------------------------------------------------------------------------------------------+

.. _dexpi-Equipment-ElectricalSeparator-DesignPower:

DesignPower
===========

.. rubric:: Attribute (data)


The power for which the :doc:`ElectricalSeparator </reference/Equipment/ElectricalSeparator>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullablePower </reference/PhysicalQuantities/NullablePower>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN POWER <http://sandbox.dexpi.org/rdl/DesignPower>`__

   Name
       DesignPower

   AttributeURI
       `http://sandbox.dexpi.org/rdl/DesignPower <http://sandbox.dexpi.org/rdl/DesignPower>`__

.. admonition:: example

   The instance electricalSeparator1 represents an :doc:`ElectricalSeparator </reference/Equipment/ElectricalSeparator>` with a :ref:`DesignPower <dexpi-Equipment-ElectricalSeparator-DesignPower>` of 500.0 kW.

   .. image:: /diagrams/diagram259.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="electricalSeparator1"
            ComponentClass="ElectricalSeparator"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/ElectricalSeparator" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="DesignPower"
              AttributeURI="http://sandbox.dexpi.org/rdl/DesignPower"
              Format="double"
              Value="500.0"
              Units="Kilowatt"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1330919" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
