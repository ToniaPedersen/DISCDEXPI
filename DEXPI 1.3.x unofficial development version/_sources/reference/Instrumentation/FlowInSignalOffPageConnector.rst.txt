============================
FlowInSignalOffPageConnector
============================

Overview
========

.. rubric:: Class

A signal connector that indicates that a preceding part of a signal conveying function is represented somewhere else, either on the same PID, or on some other PID.

.. image:: FlowInSignalOffPageConnector.*


.. rubric:: Supertypes

* :doc:`SignalConveyingFunctionSource </reference/Instrumentation/SignalConveyingFunctionSource>`

* :doc:`SignalOffPageConnector </reference/Instrumentation/SignalOffPageConnector>`
