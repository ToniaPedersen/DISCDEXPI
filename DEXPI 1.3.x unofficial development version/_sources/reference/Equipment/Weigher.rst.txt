.. _dexpi-Equipment-Weigher-DesignMassFlowRate:

DesignMassFlowRate
==================

.. rubric:: Attribute (data)


The mass flow rate for which the :doc:`Weigher </reference/Equipment/Weigher>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullableMassFlowRate </reference/PhysicalQuantities/NullableMassFlowRate>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN MASS FLOW RATE <http://data.posccaesar.org/rdl/RDS14286182>`__

   Name
       DesignMassFlowRate

   AttributeURI
       `http://data.posccaesar.org/rdl/RDS14286182 <http://data.posccaesar.org/rdl/RDS14286182>`__

.. admonition:: example

   The instance weigher1 represents a :doc:`Weigher </reference/Equipment/Weigher>` with a :ref:`DesignMassFlowRate <dexpi-Equipment-Weigher-DesignMassFlowRate>` of 420.0 kg/s.

   .. image:: /diagrams/diagram503.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="weigher1"
            ComponentClass="Weigher"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/Weigher" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="DesignMassFlowRate"
              AttributeURI="http://data.posccaesar.org/rdl/RDS14286182"
              Format="double"
              Value="420.0"
              Units="KilogramPerSecond"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1329659" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
