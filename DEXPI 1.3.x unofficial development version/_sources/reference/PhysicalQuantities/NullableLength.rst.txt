==============
NullableLength
==============

Overview
========

.. rubric:: Abstract data type

:doc:`NullableLength </reference/PhysicalQuantities/NullableLength>` is a :ref:`simple physical quantity type <dexpi-concept-simple-physical-quantity-type>` for the dimension L. :doc:`NullableLength </reference/PhysicalQuantities/NullableLength>` is abstract and has two concrete subtypes:

* a :doc:`Length </reference/PhysicalQuantities/Length>` is an :ref:`actual value <dexpi-concept-actual-value>` for a physical quantity with a numerical value and a unit of measurement;

* a :doc:`NullLength </reference/PhysicalQuantities/NullLength>` is a :ref:`null value <dexpi-concept-null-value>` that explicitly indicates the absence of an actual physical quantity.

.. image:: NullableLength.*


.. rubric:: Subtypes

* :doc:`Length </reference/PhysicalQuantities/Length>`

* :doc:`NullLength </reference/PhysicalQuantities/NullLength>`

.. admonition:: proteus

   All data attributes with type :doc:`NullableLength </reference/PhysicalQuantities/NullableLength>` are implemented as :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.
