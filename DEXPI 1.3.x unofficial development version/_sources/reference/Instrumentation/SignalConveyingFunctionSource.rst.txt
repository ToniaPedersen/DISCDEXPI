=============================
SignalConveyingFunctionSource
=============================

Overview
========

.. rubric:: Abstract class

An object than can act as the :ref:`Source <dexpi-Instrumentation-SignalConveyingFunction-Source>` of a :doc:`SignalConveyingFunction </reference/Instrumentation/SignalConveyingFunction>`.

.. image:: SignalConveyingFunctionSource.*


.. rubric:: Subtypes

* :doc:`ActuatingFunction </reference/Instrumentation/ActuatingFunction>`

* :doc:`FlowInSignalOffPageConnector </reference/Instrumentation/FlowInSignalOffPageConnector>`

* :doc:`ProcessInstrumentationFunction </reference/Instrumentation/ProcessInstrumentationFunction>`

* :doc:`ProcessSignalGeneratingFunction </reference/Instrumentation/ProcessSignalGeneratingFunction>`

.. admonition:: proteus

   Implementation is subclass-specific.
