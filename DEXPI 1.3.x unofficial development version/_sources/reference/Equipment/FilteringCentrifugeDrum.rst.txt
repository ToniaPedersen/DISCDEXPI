Chamber
=======

.. rubric:: Attribute (reference)


The :doc:`Chamber </reference/Equipment/Chamber>` in which the :doc:`FilteringCentrifugeDrum </reference/Equipment/FilteringCentrifugeDrum>` is located, if applicable. The Chamber must be a component of the same object as the :doc:`FilteringCentrifugeDrum </reference/Equipment/FilteringCentrifugeDrum>`.

Multiplicity
    0..1

Type
    :doc:`Chamber </reference/Equipment/Chamber>`

Opposite multiplicity
    0..*

   Association type for the attribute ower
       ``"is located in"``

   Opposite association type
       ``"is the location of"``
