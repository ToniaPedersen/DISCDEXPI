======
Feeder
======

Overview
========

.. rubric:: Class

A closed fluid transporter that is a gathering line tied into a trunk line (from `http://data.15926.org/rdl/RDS300644 <http://data.15926.org/rdl/RDS300644>`__).

.. image:: Feeder.*


.. rubric:: Supertypes

* :doc:`Equipment </reference/Equipment/Equipment>`

   AttributeURI
       `http://data.posccaesar.org/rdl/RDS14286182 <http://data.posccaesar.org/rdl/RDS14286182>`__
