==================
CustomCoolingTower
==================

Overview
========

.. rubric:: Class

A :ref:`custom <dexpi-concept-custom-sibling>` :doc:`CoolingTower </reference/Equipment/CoolingTower>`, i.e., a :doc:`CoolingTower </reference/Equipment/CoolingTower>` that is not covered by any of the other subclasses of :doc:`CoolingTower </reference/Equipment/CoolingTower>` (:doc:`DryCoolingTower </reference/Equipment/DryCoolingTower>`, :doc:`SprayCooler </reference/Equipment/SprayCooler>`, or :doc:`WetCoolingTower </reference/Equipment/WetCoolingTower>`\ ).

.. image:: CustomCoolingTower.*

CoolingTowerRotor

.. rubric:: Attribute (composition)

Multiplicity
    0..1

Type
    :doc:`CoolingTowerRotor </reference/Equipment/CoolingTowerRotor>`

Opposite multiplicity
    0..1

.. admonition:: proteus

   The attribute is implemented using the :ref:`XML hierarchy of the Proteus file <proteus-concept-composition-property-xml-child>`: The  element for the attribute value (a :doc:`CoolingTowerRotor </reference/Equipment/CoolingTowerRotor>`\ ) is a child of the ``<Equipment>`` element for the attribute owner (a :doc:`CustomCoolingTower </reference/Equipment/CustomCoolingTower>`\ ).
