============
CustomHeater
============

Overview
========

.. rubric:: Class

A :ref:`custom <dexpi-concept-custom-sibling>` :doc:`Heater </reference/Equipment/Heater>`, i.e., a :doc:`Heater </reference/Equipment/Heater>` that is not covered by any of the other subclasses of :doc:`Heater </reference/Equipment/Heater>` (:doc:`Boiler </reference/Equipment/Boiler>`, :doc:`Furnace </reference/Equipment/Furnace>`, or :doc:`SteamGenerator </reference/Equipment/SteamGenerator>`\ ).

.. image:: CustomHeater.*


.. rubric:: Supertypes

* :doc:`CustomObject </reference/Customization/CustomObject>`

* :doc:`Heater </reference/Equipment/Heater>`

.. admonition:: proteus

   The class is implemented as a :ref:`Proteus XML element with RDL reference <proteus-concept-xml-element-with-rdl>`.

   Tag
       ``<Equipment>``

   RDL reference
       `CUSTOM HEATER <http://sandbox.dexpi.org/rdl/CustomHeater>`__

   ComponentClass
       CustomHeater

   ComponentClassURI
       `http://sandbox.dexpi.org/rdl/CustomHeater <http://sandbox.dexpi.org/rdl/CustomHeater>`__

   .. image:: /diagrams/diagram211.*
