===========
Temperature
===========

Overview
========

.. rubric:: Data type

An :ref:`actual value <dexpi-concept-actual-value>` for a physical quantity of type :doc:`NullableTemperature </reference/PhysicalQuantities/NullableTemperature>`, i.e., a physical quantity that has a numerical value and a unit of measurement.

.. image:: Temperature.*


.. rubric:: Supertypes

* :doc:`NullableTemperature </reference/PhysicalQuantities/NullableTemperature>`

.. rubric:: Attributes (data)

+------------------------------------------------------------------------------------+--------------+--------------------------------------------------------------------------------------------------------+
| Name                                                                               | Multiplicity | Type                                                                                                   |
+====================================================================================+==============+========================================================================================================+
| :ref:`Unit <dexpi-PhysicalQuantities-Temperature-Unit>`                            | 1            | :doc:`TemperatureUnit </reference/PhysicalQuantities/TemperatureUnit>`                                 |
+------------------------------------------------------------------------------------+--------------+--------------------------------------------------------------------------------------------------------+
| :ref:`Value <dexpi-PhysicalQuantities-Temperature-Value>`                          | 1            | :doc:`Double </reference/DataTypes/Double>`                                                            |
+------------------------------------------------------------------------------------+--------------+--------------------------------------------------------------------------------------------------------+

.. admonition:: proteus

   All data attributes with type :doc:`NullableTemperature </reference/PhysicalQuantities/NullableTemperature>` (the base type of :doc:`Temperature </reference/PhysicalQuantities/Temperature>`\ ) are implemented as :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

.. admonition:: example

   The instance temperature1 represents a :doc:`Temperature </reference/PhysicalQuantities/Temperature>` of -45.0 Â°C.

   .. image:: /diagrams/diagram617.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <GenericAttribute
          Format="double"
          Value="-45.0"
          Units="DegreeCelsius"
          UnitsURI="http://data.posccaesar.org/rdl/RDS1322684" DOTS123="DOTS123">
      

      Note that the ``<GenericAttribute>`` element must have a ``Name`` and an ``AttributeURI`` attribute. They depend on the data type attribute of the DEXPI class that owns the instance temperature1. For a complete example, see the attribute :ref:`LowerLimitDesignTemperature <dexpi-Equipment-Chamber-LowerLimitDesignTemperature>` of the DEXPI class :doc:`Chamber </reference/Equipment/Chamber>`.
