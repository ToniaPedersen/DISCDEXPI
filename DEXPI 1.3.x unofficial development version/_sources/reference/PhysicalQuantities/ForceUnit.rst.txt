=========
ForceUnit
=========

Overview
========

.. rubric:: Enumeration

A unit of measurement for a physical quantity of type :doc:`NullableForce </reference/PhysicalQuantities/NullableForce>` with :ref:`dimension <dexpi-glossary-physical-dimension>` LMT\ :sup:`-2`.

.. admonition:: proteus

   :doc:`ForceUnit </reference/PhysicalQuantities/ForceUnit>` is only used as the type of the :ref:`Unit <dexpi-PhysicalQuantities-Force-Unit>` attribute of :doc:`Force </reference/PhysicalQuantities/Force>`. :doc:`Force </reference/PhysicalQuantities/Force>` is implemented using :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`. In a ``<GenericAttribute>`` element, the :doc:`ForceUnit </reference/PhysicalQuantities/ForceUnit>` literal is given by means of its :ref:`RDL reference <proteus-concept-rdl-reference>` in the table above. The ``Units`` attribute of the element is the name of the RDL reference in camel-case; it equals the name of the literal. The ``UnitsURI`` attribute of the element is the URI of the RDL reference.
