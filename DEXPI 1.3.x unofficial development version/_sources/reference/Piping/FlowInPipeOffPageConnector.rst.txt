==========================
FlowInPipeOffPageConnector
==========================

Overview
========

.. rubric:: Class

A pipe connector that indicates that a preceding part of a piping network segment is represented somewhere else, either on the same PID, or on some other PID.

.. image:: FlowInPipeOffPageConnector.*


.. rubric:: Supertypes

* :doc:`PipeOffPageConnector </reference/Piping/PipeOffPageConnector>`

* :doc:`PipingSourceItem </reference/Piping/PipingSourceItem>`
