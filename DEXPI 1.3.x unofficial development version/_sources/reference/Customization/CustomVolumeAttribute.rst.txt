.. admonition:: example

   As :doc:`CustomAttributeOwner </reference/Customization/CustomAttributeOwner>` is abstract, we consider :doc:`PackagingSystem </reference/Equipment/PackagingSystem>` as an arbitrary concrete subclass.

   The :doc:`PackagingSystem </reference/Equipment/PackagingSystem>` packagingSystem1 has a :doc:`CustomVolumeAttribute </reference/Customization/CustomVolumeAttribute>` with :ref:`AttributeName <dexpi-Customization-CustomAttribute-AttributeName>` "VolumePerPackagingUnit" and an (arbitrary) :ref:`AttributeURI <dexpi-Customization-CustomAttribute-AttributeURI>` "http://www.example.org/cad57652d80f72"DOTANDSPACEHACKThe :ref:`Value <dexpi-Customization-CustomVolumeAttribute-Value>` is 2 l.

   .. image:: /diagrams/diagram31.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="packagingSystem1"
            ComponentClass="PackagingSystem"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/PackagingSystem" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiCustomAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="VolumePerPackagingUnit"
              AttributeURI="http://www.example.org/cad57652d80f72"
              Format="double"
              Type="Volume"
              TypeURI="http://data.posccaesar.org/rdl/RDS356444"
              Value="2"
              Units="Litre"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1331144" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
