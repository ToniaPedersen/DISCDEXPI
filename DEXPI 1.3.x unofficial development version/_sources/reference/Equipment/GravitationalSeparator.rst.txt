======================
GravitationalSeparator
======================

Overview
========

.. rubric:: Class

A fluid separator that is based on the difference in specific gravity for the substances to be separated (from `http://data.15926.org/rdl/RDS16042131 <http://data.15926.org/rdl/RDS16042131>`__).

.. image:: GravitationalSeparator.*


.. rubric:: Supertypes

* :doc:`Separator </reference/Equipment/Separator>`

DesignPower
===========

.. rubric:: Attribute (data)


The power for which the :doc:`GravitationalSeparator </reference/Equipment/GravitationalSeparator>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullablePower </reference/PhysicalQuantities/NullablePower>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN POWER <http://sandbox.dexpi.org/rdl/DesignPower>`__

   Name
       DesignPower

   AttributeURI
       `http://sandbox.dexpi.org/rdl/DesignPower <http://sandbox.dexpi.org/rdl/DesignPower>`__

.. admonition:: example

   The instance gravitationalSeparator1 represents a :doc:`GravitationalSeparator </reference/Equipment/GravitationalSeparator>` with a :ref:`DesignPower <dexpi-Equipment-GravitationalSeparator-DesignPower>` of 500.0 kW.

   .. image:: /diagrams/diagram311.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="gravitationalSeparator1"
            ComponentClass="GravitySeparator"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS16042131" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="DesignPower"
              AttributeURI="http://sandbox.dexpi.org/rdl/DesignPower"
              Format="double"
              Value="500.0"
              Units="Kilowatt"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1330919" />

          </GenericAttributes>

        </Equipment>
