====
Tank
====

Overview
========

.. rubric:: Class

A vessel intended to contain fluid for storage. Typically a receiving or collecting function for further distribution. Typically with a vertical and cylindrical or square shape and a flat or conical bottom (from `http://data.posccaesar.org/rdl/RDS445139 <http://data.posccaesar.org/rdl/RDS445139>`__).

.. image:: Tank.*

.. _dexpi-Equipment-Tank-CylinderLength:

CylinderLength
==============

.. rubric:: Attribute (data)


The cylinder length of the :doc:`Tank </reference/Equipment/Tank>`.

Multiplicity
    0..1

Type
    :doc:`NullableLength </reference/PhysicalQuantities/NullableLength>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `CYLINDER LENGTH <http://sandbox.dexpi.org/rdl/CylinderLength>`__

   Name
       CylinderLength

   AttributeURI
       `http://sandbox.dexpi.org/rdl/CylinderLength <http://sandbox.dexpi.org/rdl/CylinderLength>`__

.. admonition:: example

   The instance tank1 represents a :doc:`Tank </reference/Equipment/Tank>` with a :ref:`CylinderLength <dexpi-Equipment-Tank-CylinderLength>` of 2.0 m.

   .. image:: /diagrams/diagram478.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="tank1"
            ComponentClass="Tank"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS445139" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="CylinderLength"
              AttributeURI="http://sandbox.dexpi.org/rdl/CylinderLength"
              Format="double"
              Value="2.0"
              Units="Metre"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1332674" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
