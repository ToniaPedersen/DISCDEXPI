=======
Turbine
=======

Overview
========

.. rubric:: Class

An object that is a rotary mechanical device that extracts energy from a fluid flow and converts it into useful work (from `http://data.15926.org/rdl/RDS313289 <http://data.15926.org/rdl/RDS313289>`__).

.. image:: Turbine.*


.. rubric:: Supertypes

* :doc:`Equipment </reference/Equipment/Equipment>`

* :doc:`CustomTurbine </reference/Equipment/CustomTurbine>`

* :doc:`GasTurbine </reference/Equipment/GasTurbine>`

* :doc:`SteamTurbine </reference/Equipment/SteamTurbine>`

DesignPower

Multiplicity
    0..1

Type
    :doc:`NullablePower </reference/PhysicalQuantities/NullablePower>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN POWER <http://sandbox.dexpi.org/rdl/DesignPower>`__

   Name
       DesignPower

   AttributeURI
       `http://sandbox.dexpi.org/rdl/DesignPower <http://sandbox.dexpi.org/rdl/DesignPower>`__

.. admonition:: example

   The instance turbine1 represents a :doc:`Turbine </reference/Equipment/Turbine>` with a :ref:`DesignPower <dexpi-Equipment-Turbine-DesignPower>` of 500.0 kW.
