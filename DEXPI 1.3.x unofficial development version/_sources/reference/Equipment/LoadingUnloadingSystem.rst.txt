======================
LoadingUnloadingSystem
======================

Overview
========

.. rubric:: Class

A transport system that is intended for loading and/or unloading products into/from vehicles, wagons or vessels (from `http://data.posccaesar.org/rdl/RDS11525012 <http://data.posccaesar.org/rdl/RDS11525012>`__).

.. image:: LoadingUnloadingSystem.*


.. rubric:: Supertypes

* :doc:`StationaryTransportSystem </reference/Equipment/StationaryTransportSystem>`
