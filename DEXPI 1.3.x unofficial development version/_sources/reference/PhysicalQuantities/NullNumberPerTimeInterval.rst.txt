=========================
NullNumberPerTimeInterval
=========================

Overview
========

.. rubric:: Data type

A :ref:`null value <dexpi-concept-null-value>` for a physical quantity of application type :doc:`NullableNumberPerTimeInterval </reference/PhysicalQuantities/NullableNumberPerTimeInterval>`. The only instance of this singleton type is :doc:`NULL_NUMBER_PER_TIME_INTERVAL </reference/PhysicalQuantities/NULL_NUMBER_PER_TIME_INTERVAL>`.

.. image:: NullNumberPerTimeInterval.*


.. rubric:: Supertypes

* :doc:`NullableNumberPerTimeInterval </reference/PhysicalQuantities/NullableNumberPerTimeInterval>`

.. admonition:: proteus

   All data attributes with type :doc:`NullableNumberPerTimeInterval </reference/PhysicalQuantities/NullableNumberPerTimeInterval>` (the base type of :doc:`NullNumberPerTimeInterval </reference/PhysicalQuantities/NullNumberPerTimeInterval>`\ ) are implemented as :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   .. image:: /diagrams/diagram595.*
