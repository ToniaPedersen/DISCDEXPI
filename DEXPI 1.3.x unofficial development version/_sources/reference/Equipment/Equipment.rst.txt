          ...
        </Equipment>

Multiplicity
    \*

Type
    :doc:`DryingChamber </reference/Equipment/DryingChamber>`

Opposite multiplicity
    0..1

.. admonition:: proteus

   The attribute is implemented using the :ref:`XML hierarchy of the Proteus file <proteus-concept-composition-property-xml-child>`: The  element for the attribute value (a :doc:`DryingChamber </reference/Equipment/DryingChamber>`\ ) is a child of the ``<Equipment>`` element for the attribute owner (an :doc:`Equipment </reference/Equipment/Equipment>`\ ).

.. _dexpi-Equipment-Equipment-EquipmentDescription:

EquipmentDescription
====================

.. rubric:: Attribute (data)


A short description of the :doc:`Equipment </reference/Equipment/Equipment>` in natural language.

Multiplicity
    0..1

Type
    :doc:`MultiLanguageString </reference/DataTypes/MultiLanguageString>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`set of DEXPI generic attributes for multi-language string values <proteus-concept-dexpi-generic-attribute-multi-language-string>`.

   RDL reference
       `EQUIPMENT DESCRIPTION ASSIGNMENT CLASS <http://data.posccaesar.org/rdl/RDS2181987301>`__

   Name
       EquipmentDescriptionAssignmentClass

   AttributeURI
       `http://data.posccaesar.org/rdl/RDS2181987301 <http://data.posccaesar.org/rdl/RDS2181987301>`__

.. admonition:: example

   As the owning class :doc:`Equipment </reference/Equipment/Equipment>` is abstract, we consider :doc:`Vessel </reference/Equipment/Vessel>` as an arbitrary concrete subclass.

   +----------+--------------------+
   | Language | Value              |
   +==========+====================+
   | de       | Prozessgaskühler   |
   +----------+--------------------+
   | en       | process gas cooler |
   +----------+--------------------+

   (:doc:`MultiLanguageString </reference/DataTypes/MultiLanguageString>` with 2 :doc:`SingleLanguageString </reference/DataTypes/SingleLanguageString>`\ s)

   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="vessel1"
            ComponentClass="Vessel"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS414674" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="EquipmentDescriptionAssignmentClass"
              AttributeURI="http://data.posccaesar.org/rdl/RDS2181987301"
              Format="string"
              Language="de"
              Value="Prozessgaskühler" />
            <GenericAttribute
              Name="EquipmentDescriptionAssignmentClass"
              AttributeURI="http://data.posccaesar.org/rdl/RDS2181987301"
              Format="string"
              Language="en"
              Value="process gas cooler" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
