====================
TubularHeatExchanger
====================

Overview
========

.. rubric:: Class

An indirect contact heat exchanger that separates the hot and cold fluids by tubes (from `http://data.posccaesar.org/rdl/RDS13971182 <http://data.posccaesar.org/rdl/RDS13971182>`__).

.. image:: TubularHeatExchanger.*

.. _dexpi-Equipment-TubularHeatExchanger-TemaStandardType:

TemaStandardType
================

.. rubric:: Attribute (data)


The type of the :doc:`TubularHeatExchanger </reference/Equipment/TubularHeatExchanger>` according to the Tubular Exchanger Manufacturers Association, Inc. (TEMA, `http://www.tema.org <http://www.tema.org>`__)DOTANDSPACEHACKThis is a three-letter code.

Multiplicity
    0..1

Type
    :doc:`NullableString </reference/DataTypes/NullableString>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for string values <proteus-concept-dexpi-generic-attribute-string>`.

   RDL reference
       `TEMA STANDARD TYPE ASSIGNMENT CLASS <http://sandbox.dexpi.org/rdl/TemaStandardTypeAssignmentClass>`__

   Name
       TemaStandardTypeAssignmentClass

   AttributeURI
       `http://sandbox.dexpi.org/rdl/TemaStandardTypeAssignmentClass <http://sandbox.dexpi.org/rdl/TemaStandardTypeAssignmentClass>`__

.. admonition:: example

   "AEL" (:doc:`String </reference/DataTypes/String>`\ )

   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="tubularHeatExchanger1"
            ComponentClass="TubularHeatExchanger"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS13971182" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="TemaStandardTypeAssignmentClass"
              AttributeURI="http://sandbox.dexpi.org/rdl/TemaStandardTypeAssignmentClass"
              Format="string"
              Value="AEL" />
            ...
