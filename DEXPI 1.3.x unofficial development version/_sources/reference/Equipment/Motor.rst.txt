=====
Motor
=====

Overview
========

.. rubric:: Class

A driver that is powered by electricity or internal combustion (from `http://data.15926.org/rdl/RDS7191198 <http://data.15926.org/rdl/RDS7191198>`__).

.. image:: Motor.*

.. _dexpi-Equipment-Motor-NominalPower:

NominalPower
============

.. rubric:: Attribute (data)


The nominal power of the :doc:`Motor </reference/Equipment/Motor>`.

Multiplicity
    0..1

Type
    :doc:`NullablePower </reference/PhysicalQuantities/NullablePower>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `NOMINAL POWER <http://sandbox.dexpi.org/rdl/NominalPower>`__

   Name
       NominalPower

   AttributeURI
       `http://sandbox.dexpi.org/rdl/NominalPower <http://sandbox.dexpi.org/rdl/NominalPower>`__

.. admonition:: example

   The instance motor1 represents a :doc:`Motor </reference/Equipment/Motor>` with a :ref:`NominalPower <dexpi-Equipment-Motor-NominalPower>` of 400.0 kW.

   .. image:: /diagrams/diagram369.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="motor1"
            ComponentClass="Motor"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS7191198" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="NominalPower"
              AttributeURI="http://sandbox.dexpi.org/rdl/NominalPower"
              Format="double"
              Value="400.0"
              Units="Kilowatt"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1330919" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
