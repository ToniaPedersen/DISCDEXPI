.. _dexpi-PlantStructure-Site-ParentStructure:

ParentStructure
===============

.. rubric:: Attribute (reference)


A superordinate structure of which the :doc:`Site </reference/PlantStructure/Site>` is a part.

Multiplicity
    0..1

Type
    :doc:`Enterprise </reference/PlantStructure/Enterprise>`

Opposite multiplicity
    0..*

.. admonition:: proteus

   The attribute is implemented using :ref:`Proteus \<Association\> elements <proteus-concept-reference-property-association>`.

   Association type for the attribute ower
       ``"is a part of"``

   Opposite association type
       ``"is a collection including"``

.. admonition:: example

   .. image:: /diagrams/diagram751.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <PlantStructureItem
            ID="site1"
            ComponentClass="SiteIsa95"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS10418236632" DOTS123="DOTS123">
          ...
          <Association
            Type="is a part of"
            ItemID="enterprise1" />
          ...
        <PlantStructureItem />
        ...
        <PlantStructureItem
            ID="enterprise1"
            ComponentClass="Isa95Enterprise"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS10418236543" DOTS123="DOTS123">
          ...
          <Association
            Type="is a collection including"
            ItemID="site1" />
          ...
        <PlantStructureItem />
      


.. _dexpi-PlantStructure-Site-SiteIdentificationCode:

SiteIdentificationCode
======================

.. rubric:: Attribute (data)


The identification code of the site.

Multiplicity
    0..1

Type
    :doc:`NullableString </reference/DataTypes/NullableString>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for string values <proteus-concept-dexpi-generic-attribute-string>`.

   RDL reference
       `SITE IDENTIFICATION CODE ASSIGNMENT CLASS <http://sandbox.dexpi.org/rdl/SiteIdentificationCodeAssignmentClass>`__

   Name
       SiteIdentificationCodeAssignmentClass

   AttributeURI
       `http://sandbox.dexpi.org/rdl/SiteIdentificationCodeAssignmentClass <http://sandbox.dexpi.org/rdl/SiteIdentificationCodeAssignmentClass>`__

.. admonition:: example

   "DC" (:doc:`String </reference/DataTypes/String>`\ )

   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <PlantStructureItem
            ID="site1"
            ComponentClass="SiteIsa95"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS10418236632" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="SiteIdentificationCodeAssignmentClass"
              AttributeURI="http://sandbox.dexpi.org/rdl/SiteIdentificationCodeAssignmentClass"
              Format="string"
              Value="DC" />
            ...
          </GenericAttributes>
          ...
        </PlantStructureItem>
