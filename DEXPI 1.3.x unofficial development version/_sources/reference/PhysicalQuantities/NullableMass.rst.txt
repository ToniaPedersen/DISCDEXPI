============
NullableMass
============

Overview
========

.. rubric:: Abstract data type

:doc:`NullableMass </reference/PhysicalQuantities/NullableMass>` is a :ref:`simple physical quantity type <dexpi-concept-simple-physical-quantity-type>` for the dimension M. :doc:`NullableMass </reference/PhysicalQuantities/NullableMass>` is abstract and has two concrete subtypes:

* a :doc:`Mass </reference/PhysicalQuantities/Mass>` is an :ref:`actual value <dexpi-concept-actual-value>` for a physical quantity with a numerical value and a unit of measurement;

* a :doc:`NullMass </reference/PhysicalQuantities/NullMass>` is a :ref:`null value <dexpi-concept-null-value>` that explicitly indicates the absence of an actual physical quantity.

.. image:: NullableMass.*


.. rubric:: Subtypes

* :doc:`Mass </reference/PhysicalQuantities/Mass>`

* :doc:`NullMass </reference/PhysicalQuantities/NullMass>`

.. admonition:: proteus

   All data attributes with type :doc:`NullableMass </reference/PhysicalQuantities/NullableMass>` are implemented as :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   See the examples for :doc:`Mass </reference/PhysicalQuantities/Mass>` and :doc:`NullMass </reference/PhysicalQuantities/NullMass>`.
