====================
NullableMassFlowRate
====================

Overview
========

.. rubric:: Abstract data type

:doc:`NullableMassFlowRate </reference/PhysicalQuantities/NullableMassFlowRate>` is a :ref:`simple physical quantity type <dexpi-concept-simple-physical-quantity-type>` for the dimension MT\ :sup:`-1`. :doc:`NullableMassFlowRate </reference/PhysicalQuantities/NullableMassFlowRate>` is abstract and has two concrete subtypes:

* a :doc:`MassFlowRate </reference/PhysicalQuantities/MassFlowRate>` is an :ref:`actual value <dexpi-concept-actual-value>` for a physical quantity with a numerical value and a unit of measurement;

* a :doc:`NullMassFlowRate </reference/PhysicalQuantities/NullMassFlowRate>` is a :ref:`null value <dexpi-concept-null-value>` that explicitly indicates the absence of an actual physical quantity.

.. rubric:: Subtypes

* :doc:`MassFlowRate </reference/PhysicalQuantities/MassFlowRate>`
