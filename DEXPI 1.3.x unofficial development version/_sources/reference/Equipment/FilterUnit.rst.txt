.. _dexpi-Equipment-FilterUnit-Chamber:

Chamber
=======

.. rubric:: Attribute (reference)


The :doc:`Chamber </reference/Equipment/Chamber>` in which the :doc:`FilterUnit </reference/Equipment/FilterUnit>` is located, if applicable. The Chamber must be a component of the same object as the FilterUnit.

Multiplicity
    0..1

Type
    :doc:`Chamber </reference/Equipment/Chamber>`

Opposite multiplicity
    0..*

.. admonition:: proteus

   The attribute is implemented using :ref:`Proteus \<Association\> elements <proteus-concept-reference-property-association>`.

   Association type for the attribute ower
       ``"is located in"``

   Opposite association type
       ``"is the location of"``

.. admonition:: example

   .. image:: /diagrams/diagram281.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="filterUnit1"
            ComponentClass="FilterUnit"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/FilterUnit" DOTS123="DOTS123">
          ...
          <Association
            Type="is located in"
            ItemID="chamber1" />
          ...
        <Equipment />
        ...
        <Equipment
            ID="chamber1"
            ComponentClass="Chamber"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS903151421" DOTS123="DOTS123">
          ...
          <Association
            Type="is the location of"
            ItemID="filterUnit1" />
          ...
        <Equipment />
