====
Lift
====

Overview
========

.. rubric:: Class

A :doc:`StationaryTransportSystem </reference/Equipment/StationaryTransportSystem>` for transporting persons or things from one level to another (from `http://data.posccaesar.org/rdl/RDS13601120 <http://data.posccaesar.org/rdl/RDS13601120>`__).

.. image:: Lift.*


.. rubric:: Supertypes

* :doc:`StationaryTransportSystem </reference/Equipment/StationaryTransportSystem>`

.. _dexpi-Equipment-Lift-DischargeHead:

DischargeHead
=============

.. rubric:: Attribute (data)


The length of the :doc:`Lift </reference/Equipment/Lift>`.

Multiplicity
    0..1

Type
    :doc:`NullableLength </reference/PhysicalQuantities/NullableLength>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DISCHARGE HEAD <http://sandbox.dexpi.org/rdl/DischargeHead>`__

   Name
       DischargeHead

   AttributeURI
       `http://sandbox.dexpi.org/rdl/DischargeHead <http://sandbox.dexpi.org/rdl/DischargeHead>`__

.. admonition:: example

   The instance lift1 represents a :doc:`Lift </reference/Equipment/Lift>` with a :ref:`DischargeHead <dexpi-Equipment-Lift-DischargeHead>` of 2.0 m.

   .. image:: /diagrams/diagram339.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="lift1"
            ComponentClass="Lift"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS13601120" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="DischargeHead"
              AttributeURI="http://sandbox.dexpi.org/rdl/DischargeHead"
              Format="double"
              Value="2.0"
              Units="Metre"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1332674" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
