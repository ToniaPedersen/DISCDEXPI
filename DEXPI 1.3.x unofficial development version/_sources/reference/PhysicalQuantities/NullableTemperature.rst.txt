===================
NullableTemperature
===================

Overview
========

.. rubric:: Abstract data type

:doc:`NullableTemperature </reference/PhysicalQuantities/NullableTemperature>` is a :ref:`simple physical quantity type <dexpi-concept-simple-physical-quantity-type>` for the dimension Î˜. :doc:`NullableTemperature </reference/PhysicalQuantities/NullableTemperature>` is abstract and has two concrete subtypes:

* a :doc:`Temperature </reference/PhysicalQuantities/Temperature>` is an :ref:`actual value <dexpi-concept-actual-value>` for a physical quantity with a numerical value and a unit of measurement;

* a :doc:`NullTemperature </reference/PhysicalQuantities/NullTemperature>` is a :ref:`null value <dexpi-concept-null-value>` that explicitly indicates the absence of an actual physical quantity.

.. rubric:: Subtypes
