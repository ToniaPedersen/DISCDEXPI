.. admonition:: proteus

   All data attributes with type :doc:`NominalPressureStandardClassification </reference/Enumerations/NominalPressureStandardClassification>` are implemented as :ref:`DEXPI generic attributes for enumeration values <proteus-concept-dexpi-generic-attribute-enumeration>`. In a ``<GenericAttribute>`` element, the :doc:`NominalPressureStandardClassification </reference/Enumerations/NominalPressureStandardClassification>` literal is given by means of its :ref:`RDL reference <proteus-concept-rdl-reference>` in the table above. The ``Value`` attribute of the element is the name of the RDL reference in camel-case; it equals the name of the literal. The ``ValueURI`` attribute of the element is the URI of the RDL reference.

.. admonition:: example

   .. image:: /diagrams/diagram62.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <GenericAttribute
          Format="anyURI"
          Value="En1333Pn40Artefact"
          ValueURI="http://sandbox.dexpi.org/rdl/En1333Pn40Artefact" DOTS123="DOTS123">
      

      Note that the ``<GenericAttribute>`` element must have a ``Name`` and an ``AttributeURI`` attribute. They depend on the data type attribute of the DEXPI class that owns the instance :ref:`En1333Pn40Artefact <dexpi-Enumerations-NominalPressureStandardClassification-En1333Pn40Artefact>`. For a complete example, see the attribute :ref:`NominalPressureStandard <dexpi-Equipment-Nozzle-NominalPressureStandard>` of the DEXPI class :doc:`Nozzle </reference/Equipment/Nozzle>`.
