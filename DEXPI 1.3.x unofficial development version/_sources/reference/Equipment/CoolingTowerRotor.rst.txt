.. _dexpi-Equipment-CoolingTowerRotor-Chamber:

Chamber
=======

.. rubric:: Attribute (reference)


The :doc:`Chamber </reference/Equipment/Chamber>` in which the :doc:`CoolingTowerRotor </reference/Equipment/CoolingTowerRotor>` is located, if applicable. The Chamber must be a component of the same object as the :doc:`CoolingTowerRotor </reference/Equipment/CoolingTowerRotor>`.

Multiplicity
    0..1

Type
    :doc:`Chamber </reference/Equipment/Chamber>`

Opposite multiplicity
    0..*

.. admonition:: proteus

   The attribute is implemented using :ref:`Proteus \<Association\> elements <proteus-concept-reference-property-association>`.

   Association type for the attribute ower
       ``"is located in"``

   Opposite association type
       ``"is the location of"``

.. admonition:: example

   .. image:: /diagrams/diagram179.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="coolingTowerRotor1"
            ComponentClass="CoolingTowerRotor"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/CoolingTowerRotor" DOTS123="DOTS123">
          ...
          <Association
            Type="is located in"
            ItemID="chamber1" />
          ...
        <Equipment />
        ...
        <Equipment
            ID="chamber1"
            ComponentClass="Chamber"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS903151421" DOTS123="DOTS123">
          ...
          <Association
            Type="is the location of"
            ItemID="coolingTowerRotor1" />
          ...
        <Equipment />
      


.. _dexpi-Equipment-CoolingTowerRotor-Diameter:

Diameter
========

.. rubric:: Attribute (data)


The diameter of the :doc:`CoolingTowerRotor </reference/Equipment/CoolingTowerRotor>`.

Multiplicity
    0..1

Type
    :doc:`NullableLength </reference/PhysicalQuantities/NullableLength>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DIAMETER <http://data.posccaesar.org/rdl/RDS350954>`__

   Name
       Diameter

   AttributeURI
       `http://data.posccaesar.org/rdl/RDS350954 <http://data.posccaesar.org/rdl/RDS350954>`__

.. admonition:: example

   The instance coolingTowerRotor1 represents a :doc:`CoolingTowerRotor </reference/Equipment/CoolingTowerRotor>` with a :ref:`Diameter <dexpi-Equipment-CoolingTowerRotor-Diameter>` of 20.0 cm.

   .. image:: /diagrams/diagram180.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="coolingTowerRotor1"
            ComponentClass="CoolingTowerRotor"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/CoolingTowerRotor" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="Diameter"
              AttributeURI="http://data.posccaesar.org/rdl/RDS350954"
              Format="double"
              Value="20.0"
              Units="Centimetre"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1318004" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
