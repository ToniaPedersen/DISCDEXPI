======
String
======

Overview
========

.. rubric:: Data type

A sequence of characters. The value space of :doc:`String </reference/DataTypes/String>` is the same as that of the XML Schema data type `string <https://www.w3.org/TR/xmlschema-2/#string>`__ as specified by the W3C Recommendation `XML Schema Part 2: Datatypes Second Edition <https://www.w3.org/TR/xmlschema-2>`__ from October 28, 2004.

.. image:: String.*


.. rubric:: Supertypes

* :doc:`NullableString </reference/DataTypes/NullableString>`

.. admonition:: proteus

   A :doc:`String </reference/DataTypes/String>` is implemented as a literal for the XML Schema data type `string <https://www.w3.org/TR/xmlschema-2/#string>`__.

   The way this literal is used in a Proteus XML document depends on the Proteus Schema implementations of the data attributes with type :doc:`NullableString </reference/DataTypes/NullableString>` or :doc:`String </reference/DataTypes/String>`.

   * :doc:`NullableString </reference/DataTypes/NullableString>` is the type of the :ref:`Value <dexpi-Customization-CustomStringAttribute-Value>` attribute of :doc:`CustomStringAttribute </reference/Customization/CustomStringAttribute>`. :doc:`CustomStringAttribute </reference/Customization/CustomStringAttribute>` is implemented as a :ref:`DEXPI custom generic attribute <proteus-concept-dexpi-custom-generic-attribute>`.

   * All other attributes with type :doc:`NullableString </reference/DataTypes/NullableString>` are implemented as :ref:`DEXPI generic attributes <proteus-concept-dexpi-generic-attribute>`. For example, see the :ref:`MaterialOfConstructionCode <dexpi-Equipment-AgitatorRotor-MaterialOfConstructionCode>` attribute of :doc:`AgitatorRotor </reference/Equipment/AgitatorRotor>`.

   * For the attributes with direct type :doc:`String </reference/DataTypes/String>` special rules apply. These attributes are:

     * :doc:`AttributeRepresentation </reference/Graphics/AttributeRepresentation>`:::ref:`AttributeName <dexpi-Graphics-AttributeRepresentation-AttributeName>`

     * :doc:`CustomAttribute </reference/Customization/CustomAttribute>`:::ref:`AttributeName <dexpi-Customization-CustomAttribute-AttributeName>`

     * :doc:`CustomObject </reference/Customization/CustomObject>`:::ref:`TypeName <dexpi-Customization-CustomObject-TypeName>`

     * :doc:`Diagram </reference/Graphics/Diagram>`:::ref:`Name <dexpi-Graphics-Diagram-Name>`

     * :doc:`LiteralText </reference/Graphics/LiteralText>`:::ref:`Text <dexpi-Graphics-LiteralText-Text>`

     * :doc:`Shape </reference/Graphics/Shape>`:::ref:`Name <dexpi-Graphics-Shape-Name>`

     * :doc:`Shape </reference/Graphics/Shape>`:::ref:`SymbolRegistrationNumber <dexpi-Graphics-Shape-SymbolRegistrationNumber>`

     * :doc:`ShapeCatalogue </reference/Graphics/ShapeCatalogue>`:::ref:`Name <dexpi-Graphics-ShapeCatalogue-Name>`

     * :doc:`Text </reference/Graphics/Text>`:::ref:`Font <dexpi-Graphics-Text-Font>`

     * :doc:`Text </reference/Graphics/Text>`:::ref:`Text <dexpi-Graphics-Text-Text>`
