.. admonition:: proteus

   All data attributes with type :doc:`NominalDiameterBreakClassification </reference/Enumerations/NominalDiameterBreakClassification>` are implemented as :ref:`DEXPI generic attributes for enumeration values <proteus-concept-dexpi-generic-attribute-enumeration>`. In a ``<GenericAttribute>`` element, the :doc:`NominalDiameterBreakClassification </reference/Enumerations/NominalDiameterBreakClassification>` literal is given by means of its :ref:`RDL reference <proteus-concept-rdl-reference>` in the table above. The ``Value`` attribute of the element is the name of the RDL reference in camel-case; it equals the name of the literal. The ``ValueURI`` attribute of the element is the URI of the RDL reference.
