.. _dexpi-Equipment-HeatExchangerRotor-Chamber:

Chamber
=======

.. rubric:: Attribute (reference)


The :doc:`Chamber </reference/Equipment/Chamber>` in which the :doc:`HeatExchangerRotor </reference/Equipment/HeatExchangerRotor>` is located, if applicable. The Chamber must be a component of the same object as the HeatExchangerRotor.

Multiplicity
    0..1

Type
    :doc:`Chamber </reference/Equipment/Chamber>`

Opposite multiplicity
    0..*

.. admonition:: proteus

   The attribute is implemented using :ref:`Proteus \<Association\> elements <proteus-concept-reference-property-association>`.

   Association type for the attribute ower
       ``"is located in"``

   Opposite association type
       ``"is the location of"``

.. admonition:: example

   .. image:: /diagrams/diagram322.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="heatExchangerRotor1"
            ComponentClass="HeatExchangerRotor"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/HeatExchangerRotor" DOTS123="DOTS123">
          ...
          <Association
            Type="is located in"
            ItemID="chamber1" />
          ...
        <Equipment />
        ...
        <Equipment
            ID="chamber1"
            ComponentClass="Chamber"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS903151421" DOTS123="DOTS123">
          ...
          <Association
            Type="is the location of"
            ItemID="heatExchangerRotor1" />
          ...
        <Equipment />
      


.. _dexpi-Equipment-HeatExchangerRotor-MaterialOfConstructionCode:

MaterialOfConstructionCode
==========================

.. rubric:: Attribute (data)


A code that gives the material of construction of the :doc:`HeatExchangerRotor </reference/Equipment/HeatExchangerRotor>`.

Multiplicity
    0..1

Type
    :doc:`NullableString </reference/DataTypes/NullableString>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for string values <proteus-concept-dexpi-generic-attribute-string>`.

   RDL reference
       `MATERIAL OF CONSTRUCTION CODE ASSIGNMENT CLASS <http://data.posccaesar.org/rdl/RDS1460719741>`__

   Name
       MaterialOfConstructionCodeAssignmentClass

   AttributeURI
       `http://data.posccaesar.org/rdl/RDS1460719741 <http://data.posccaesar.org/rdl/RDS1460719741>`__

.. admonition:: example

   "1.4306" (:doc:`String </reference/DataTypes/String>`\ )

   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="heatExchangerRotor1"
            ComponentClass="HeatExchangerRotor"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/HeatExchangerRotor" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="MaterialOfConstructionCodeAssignmentClass"
              AttributeURI="http://data.posccaesar.org/rdl/RDS1460719741"
              Format="string"
              Value="1.4306" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
