.. _dexpi-Piping-PipingNode-NominalDiameterNumericalValueRepresentation:

NominalDiameterNumericalValueRepresentation
===========================================

.. rubric:: Attribute (data)


A readable representation of the numerical value of the nominal diameter of the :doc:`PipingNode </reference/Piping/PipingNode>`, without any type or unit of measure.

Multiplicity
    0..1

Type
    :doc:`NullableString </reference/DataTypes/NullableString>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for string values <proteus-concept-dexpi-generic-attribute-string>`.

   RDL reference
       `NOMINAL DIAMETER NUMERICAL VALUE REPRESENTATION ASSIGNMENT CLASS <http://sandbox.dexpi.org/rdl/NominalDiameterNumericalValueRepresentationAssignmentClass>`__

   Name
       NominalDiameterNumericalValueRepresentationAssignmentClass

   AttributeURI
       `http://sandbox.dexpi.org/rdl/NominalDiameterNumericalValueRepresentationAssignmentClass <http://sandbox.dexpi.org/rdl/NominalDiameterNumericalValueRepresentationAssignmentClass>`__

.. admonition:: example

   "25" (:doc:`String </reference/DataTypes/String>`\ )

   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Node
            ID="pipingNode1"
            Type="process">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="NominalDiameterNumericalValueRepresentationAssignmentClass"
              AttributeURI="http://sandbox.dexpi.org/rdl/NominalDiameterNumericalValueRepresentationAssignmentClass"
              Format="string"
              Value="25" />
            ...
          </GenericAttributes>
          ...
        </Node>

.. _dexpi-Piping-PipingNode-NominalDiameterStandard:

NominalDiameterStandard
=======================

.. rubric:: Attribute (data)


The nominal diameter of the :doc:`PipingNode </reference/Piping/PipingNode>`, given as a reference to a nominal diameter standard and value.

Multiplicity
    0..1

Type
    :doc:`NominalDiameterStandardClassification </reference/Enumerations/NominalDiameterStandardClassification>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for enumeration values <proteus-concept-dexpi-generic-attribute-enumeration>`.

   RDL reference
       `NOMINAL DIAMETER STANDARD SPECIALIZATION <http://sandbox.dexpi.org/rdl/NominalDiameterStandardSpecialization>`__

   Name
       NominalDiameterStandardSpecialization

   AttributeURI
       `http://sandbox.dexpi.org/rdl/NominalDiameterStandardSpecialization <http://sandbox.dexpi.org/rdl/NominalDiameterStandardSpecialization>`__

.. admonition:: example

   DN 25 (DIN 2448) (:ref:`NominalDiameterStandardClassification::Din2448ObjectDn25 <dexpi-Enumerations-NominalDiameterStandardClassification-Din2448ObjectDn25>`\ )

   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Node
            ID="pipingNode1"
            Type="process">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="NominalDiameterStandardSpecialization"
              AttributeURI="http://sandbox.dexpi.org/rdl/NominalDiameterStandardSpecialization"
              Format="anyURI"
              Value="Din2448ObjectDn25"
              ValueURI="http://sandbox.dexpi.org/rdl/Din2448ObjectDn25" />
            ...
          </GenericAttributes>
          ...
        </Node>
