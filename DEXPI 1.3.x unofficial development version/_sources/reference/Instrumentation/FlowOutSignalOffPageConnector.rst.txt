=============================
FlowOutSignalOffPageConnector
=============================

Overview
========

.. rubric:: Class

A signal connector that indicates that a subsequent part of a signal conveying function is represented somewhere else, either on the same PID, or on some other PID.

.. image:: FlowOutSignalOffPageConnector.*


.. rubric:: Supertypes

* :doc:`SignalConveyingFunctionTarget </reference/Instrumentation/SignalConveyingFunctionTarget>`

* :doc:`SignalOffPageConnector </reference/Instrumentation/SignalOffPageConnector>`
