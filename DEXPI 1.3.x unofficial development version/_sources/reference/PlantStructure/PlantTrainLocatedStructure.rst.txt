.. _dexpi-PlantStructure-PlantTrainLocatedStructure-PlantTrain:

PlantTrain
==========

.. rubric:: Attribute (reference)


The :doc:`PlantTrain </reference/PlantStructure/PlantTrain>` in which the :doc:`PlantTrainLocatedStructure </reference/PlantStructure/PlantTrainLocatedStructure>` is located.

Multiplicity
    0..1

Type
    :doc:`PlantTrain </reference/PlantStructure/PlantTrain>`

Opposite multiplicity
    0..*

.. admonition:: proteus

   The attribute is implemented using :ref:`Proteus \<Association\> elements <proteus-concept-reference-property-association>`.

   Association type for the attribute ower
       ``"is located in"``

   Opposite association type
       ``"is the location of"``

.. admonition:: example

   .. image:: /diagrams/diagram746.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="tank1"
            ComponentClass="Tank"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS445139" DOTS123="DOTS123">
          ...
          <Association
            Type="is located in"
            ItemID="plantTrain1" />
          ...
        <Equipment />
        ...
        <PlantStructureItem
            ID="plantTrain1"
            ComponentClass="PlantTrain"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/PlantTrain" DOTS123="DOTS123">
          ...
          <Association
            Type="is the location of"
            ItemID="tank1" />
          ...
        <PlantStructureItem />
