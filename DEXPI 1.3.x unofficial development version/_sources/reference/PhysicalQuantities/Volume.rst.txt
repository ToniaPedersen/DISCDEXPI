======
Volume
======

Overview
========

.. rubric:: Data type

An :ref:`actual value <dexpi-concept-actual-value>` for a physical quantity of type :doc:`NullableVolume </reference/PhysicalQuantities/NullableVolume>`, i.e., a physical quantity that has a numerical value and a unit of measurement.

.. image:: Volume.*


.. rubric:: Supertypes

* :doc:`NullableVolume </reference/PhysicalQuantities/NullableVolume>`

.. rubric:: Attributes (data)

+--------------------------------------------------------------------------------+--------------+----------------------------------------------------------------------------------------+
| Name                                                                           | Multiplicity | Type                                                                                   |
+================================================================================+==============+========================================================================================+
| :ref:`Unit <dexpi-PhysicalQuantities-Volume-Unit>`                             | 1            | :doc:`VolumeUnit </reference/PhysicalQuantities/VolumeUnit>`                           |
+--------------------------------------------------------------------------------+--------------+----------------------------------------------------------------------------------------+
| :ref:`Value <dexpi-PhysicalQuantities-Volume-Value>`                           | 1            | :doc:`Double </reference/DataTypes/Double>`                                            |
+--------------------------------------------------------------------------------+--------------+----------------------------------------------------------------------------------------+

.. admonition:: proteus

   All data attributes with type :doc:`NullableVolume </reference/PhysicalQuantities/NullableVolume>` (the base type of :doc:`Volume </reference/PhysicalQuantities/Volume>`\ ) are implemented as :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

.. admonition:: example

   The instance volume1 represents a :doc:`Volume </reference/PhysicalQuantities/Volume>` of 80.0 cm\ :sup:`3`.

   .. image:: /diagrams/diagram621.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <GenericAttribute
          Format="double"
          Value="80.0"
          Units="CentimetreCubed"
          UnitsURI="http://data.posccaesar.org/rdl/RDS1357874" DOTS123="DOTS123">
      

      Note that the ``<GenericAttribute>`` element must have a ``Name`` and an ``AttributeURI`` attribute. They depend on the data type attribute of the DEXPI class that owns the instance volume1. For a complete example, see the attribute :ref:`VolumePerStroke <dexpi-Equipment-Displacer-VolumePerStroke>` of the DEXPI class :doc:`Displacer </reference/Equipment/Displacer>`.
