======
Nozzle
======

Overview
========

.. rubric:: Class

A physical object that has a protruding part through which a stream of fluid is directed (from `http://data.posccaesar.org/rdl/RDS415214 <http://data.posccaesar.org/rdl/RDS415214>`__).

.. image:: Nozzle.*

.. _dexpi-Equipment-Nozzle-Chamber:

Chamber
=======

.. rubric:: Attribute (reference)


The :doc:`Chamber </reference/Equipment/Chamber>` at which the :doc:`Nozzle </reference/Equipment/Nozzle>` is located, if applicable. The Chamber must be a component of the same object as the Nozzle.

Multiplicity
    0..1

Type
    :doc:`Chamber </reference/Equipment/Chamber>`

Opposite multiplicity
    0..*

.. admonition:: proteus

   The attribute is implemented using :ref:`Proteus \<Association\> elements <proteus-concept-reference-property-association>`.

   Association type for the attribute ower
       ``"is located in"``

   Opposite association type
       ``"is the location of"``

.. admonition:: example

   .. image:: /diagrams/diagram375.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Nozzle
            ID="nozzle1"
            ComponentClass="Nozzle"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS415214" DOTS123="DOTS123">
          ...
          <Association
            Type="is located in"
            ItemID="chamber1" />
          ...
        <Nozzle />
        ...
