===========================
FlowOutPipeOffPageConnector
===========================

Overview
========

.. rubric:: Class

A pipe connector that indicates that a subsequent part of a piping network segment is represented somewhere else, either on the same PID, or on some other PID.

.. image:: FlowOutPipeOffPageConnector.*


.. rubric:: Supertypes

* :doc:`PipeOffPageConnector </reference/Piping/PipeOffPageConnector>`

* :doc:`PipingTargetItem </reference/Piping/PipingTargetItem>`

   RDL reference
