.. _dexpi-Customization-CustomObject-TypeName:

TypeName
========

.. rubric:: Attribute (data)


A name that identifies the type of the :doc:`CustomObject </reference/Customization/CustomObject>`.

Multiplicity
    1

Type
    :doc:`String </reference/DataTypes/String>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for string values <proteus-concept-dexpi-generic-attribute-string>`.

   RDL reference
       `TYPE NAME ASSIGNMENT CLASS <http://sandbox.dexpi.org/rdl/TypeNameAssignmentClass>`__

   Name
       TypeNameAssignmentClass

   AttributeURI
       `http://sandbox.dexpi.org/rdl/TypeNameAssignmentClass <http://sandbox.dexpi.org/rdl/TypeNameAssignmentClass>`__

.. admonition:: example

   "micro impedance pump" (:doc:`String </reference/DataTypes/String>`\ )

   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="customPump1"
            ComponentClass="CustomPump"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/CustomPump" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="TypeNameAssignmentClass"
              AttributeURI="http://sandbox.dexpi.org/rdl/TypeNameAssignmentClass"
              Format="string"
              Value="micro impedance pump" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
      


.. _dexpi-Customization-CustomObject-TypeURI:

TypeURI
=======

.. rubric:: Attribute (data)


A URI that identifies the type of the :doc:`CustomObject </reference/Customization/CustomObject>`.

Multiplicity
    1

Type
    :doc:`NullableAnyURI </reference/DataTypes/NullableAnyURI>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for URI values <proteus-concept-dexpi-generic-attribute-any-uri>`.

   RDL reference
       `TYPE URI ASSIGNMENT CLASS <http://sandbox.dexpi.org/rdl/TypeURIAssignmentClass>`__

   Name
       TypeURIAssignmentClass

   AttributeURI
       `http://sandbox.dexpi.org/rdl/TypeURIAssignmentClass <http://sandbox.dexpi.org/rdl/TypeURIAssignmentClass>`__

.. admonition:: example

   ``http://www.example.org/MicroImpedancePump`` (:doc:`AnyURI </reference/DataTypes/AnyURI>`\ )

   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="customPump1"
            ComponentClass="CustomPump"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/CustomPump" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="TypeURIAssignmentClass"
              AttributeURI="http://sandbox.dexpi.org/rdl/TypeURIAssignmentClass"
              Format="anyURI"
              Value="http://www.example.org/MicroImpedancePump" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
