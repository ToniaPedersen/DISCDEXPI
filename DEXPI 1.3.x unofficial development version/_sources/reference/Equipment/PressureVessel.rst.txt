==============
PressureVessel
==============

Overview
========

.. rubric:: Class

A vessel intended to withstand external and/or internal pressure (from `http://data.posccaesar.org/rdl/RDS427229 <http://data.posccaesar.org/rdl/RDS427229>`__).

.. image:: PressureVessel.*


.. rubric:: Supertypes

* :doc:`Vessel </reference/Equipment/Vessel>`

.. rubric:: Attributes (data)

+----------------------------------------------------------------------------------------------------+--------------+----------------------------------------------------------------------------------------------------+
| Name                                                                                               | Multiplicity | Type                                                                                               |
+====================================================================================================+==============+====================================================================================================+
| :ref:`CylinderLength <dexpi-Equipment-PressureVessel-CylinderLength>`                              | 0..1         | :doc:`NullableLength </reference/PhysicalQuantities/NullableLength>`                               |
+----------------------------------------------------------------------------------------------------+--------------+----------------------------------------------------------------------------------------------------+

   ComponentClassURI

.. _dexpi-Equipment-PressureVessel-CylinderLength:

CylinderLength
==============

.. rubric:: Attribute (data)


The cylinder length of the :doc:`PressureVessel </reference/Equipment/PressureVessel>`.

Multiplicity
    0..1

Type
    :doc:`NullableLength </reference/PhysicalQuantities/NullableLength>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `CYLINDER LENGTH <http://sandbox.dexpi.org/rdl/CylinderLength>`__

   Name
       CylinderLength

   AttributeURI
       `http://sandbox.dexpi.org/rdl/CylinderLength <http://sandbox.dexpi.org/rdl/CylinderLength>`__

.. admonition:: example

   The instance pressureVessel1 represents a :doc:`PressureVessel </reference/Equipment/PressureVessel>` with a :ref:`CylinderLength <dexpi-Equipment-PressureVessel-CylinderLength>` of 2.0 m.

   .. image:: /diagrams/diagram388.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="pressureVessel1"
            ComponentClass="PressureVessel"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS427229" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="CylinderLength"
              AttributeURI="http://sandbox.dexpi.org/rdl/CylinderLength"
              Format="double"
              Value="2.0"
              Units="Metre"
              UnitsURI="http://data.posccaesar.org/rdl/RDS1332674" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
