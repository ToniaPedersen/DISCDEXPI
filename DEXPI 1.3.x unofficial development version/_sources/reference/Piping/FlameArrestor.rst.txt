=============
FlameArrestor
=============

Overview
========

.. rubric:: Class

An 'arrestor' which is a trap covering an opening, e.g of a ventilation system or a pipe, to prevent flames from entering the system (from `http://data.posccaesar.org/rdl/RDS1325028651 <http://data.posccaesar.org/rdl/RDS1325028651>`__).

.. image:: FlameArrestor.*
