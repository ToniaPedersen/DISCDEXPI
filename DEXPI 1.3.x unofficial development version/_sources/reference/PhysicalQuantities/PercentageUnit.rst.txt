==============
PercentageUnit
==============

Overview
========

.. rubric:: Enumeration

A unit of measurement for a quantity of type :doc:`NullablePercentage </reference/PhysicalQuantities/NullablePercentage>`.

.. admonition:: proteus

   :doc:`PercentageUnit </reference/PhysicalQuantities/PercentageUnit>` is only used as the type of the :ref:`Unit <dexpi-PhysicalQuantities-Percentage-Unit>` attribute of :doc:`Percentage </reference/PhysicalQuantities/Percentage>`. :doc:`Percentage </reference/PhysicalQuantities/Percentage>` is implemented using :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`. In a ``<GenericAttribute>`` element, the :doc:`PercentageUnit </reference/PhysicalQuantities/PercentageUnit>` literal is given by means of its :ref:`RDL reference <proteus-concept-rdl-reference>` in the table above. The ``Units`` attribute of the element is the name of the RDL reference in camel-case; it equals the name of the literal. The ``UnitsURI`` attribute of the element is the URI of the RDL reference.
