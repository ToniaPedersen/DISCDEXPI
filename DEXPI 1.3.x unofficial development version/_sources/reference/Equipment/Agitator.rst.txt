========
Agitator
========

Overview
========

.. rubric:: Class

An Agitator is a dynamic mixer that stirs or shakes fluids by reaction force from moving vanes.

.. image:: Agitator.*


.. rubric:: Supertypes

* :doc:`Equipment </reference/Equipment/Equipment>`

.. _dexpi-Equipment-Agitator-DesignRotationalSpeed:

DesignRotationalSpeed
=====================

.. rubric:: Attribute (data)


The rotational speed for which the :doc:`Agitator </reference/Equipment/Agitator>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullableRotationalFrequency </reference/PhysicalQuantities/NullableRotationalFrequency>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN ROTATIONAL SPEED <http://sandbox.dexpi.org/rdl/DesignRotationalSpeed>`__

   Name
       DesignRotationalSpeed

   AttributeURI
       `http://sandbox.dexpi.org/rdl/DesignRotationalSpeed <http://sandbox.dexpi.org/rdl/DesignRotationalSpeed>`__

.. admonition:: example

   The instance agitator1 represents an :doc:`Agitator </reference/Equipment/Agitator>` with a :ref:`DesignRotationalSpeed <dexpi-Equipment-Agitator-DesignRotationalSpeed>` of 180.0 min\ :sup:`-1`.

   .. image:: /diagrams/diagram83.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="agitator1"
            ComponentClass="Agitator"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS16045622" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="DesignRotationalSpeed"
              AttributeURI="http://sandbox.dexpi.org/rdl/DesignRotationalSpeed"
              Format="double"
              Value="180.0"
              Units="ReciprocalMinute"
              UnitsURI="http://data.posccaesar.org/rdl/RDS4316851589" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
