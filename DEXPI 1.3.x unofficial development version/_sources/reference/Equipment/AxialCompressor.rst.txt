===============
AxialCompressor
===============

Overview
========

.. rubric:: Class

A :doc:`Compressor </reference/Equipment/Compressor>` in which the gas is accelerated by the action of a bladed rotor and where the main flow is along the rotation axis of the rotor.

.. image:: AxialCompressor.*


.. rubric:: Supertypes

* :doc:`Compressor </reference/Equipment/Compressor>`

.. _dexpi-Equipment-AxialCompressor-DesignRotationalSpeed:

DesignRotationalSpeed
=====================

.. rubric:: Attribute (data)


The rotational speed for which the :doc:`AxialCompressor </reference/Equipment/AxialCompressor>` is designed.

Multiplicity
    0..1

Type
    :doc:`NullableRotationalFrequency </reference/PhysicalQuantities/NullableRotationalFrequency>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

   RDL reference
       `DESIGN ROTATIONAL SPEED <http://sandbox.dexpi.org/rdl/DesignRotationalSpeed>`__

   Name
       DesignRotationalSpeed

   AttributeURI
       `http://sandbox.dexpi.org/rdl/DesignRotationalSpeed <http://sandbox.dexpi.org/rdl/DesignRotationalSpeed>`__

.. admonition:: example

   The instance axialCompressor1 represents an :doc:`AxialCompressor </reference/Equipment/AxialCompressor>` with a :ref:`DesignRotationalSpeed <dexpi-Equipment-AxialCompressor-DesignRotationalSpeed>` of 180.0 min\ :sup:`-1`.

   .. image:: /diagrams/diagram109.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="axialCompressor1"
            ComponentClass="AxialCompressor"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS417239" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="DesignRotationalSpeed"
              AttributeURI="http://sandbox.dexpi.org/rdl/DesignRotationalSpeed"
              Format="double"
              Value="180.0"
              Units="ReciprocalMinute"
              UnitsURI="http://data.posccaesar.org/rdl/RDS4316851589" />
            ...
          </GenericAttributes>
          ...
        </Equipment>
