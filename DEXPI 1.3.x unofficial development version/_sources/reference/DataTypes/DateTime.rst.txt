========
DateTime
========

Overview
========

.. rubric:: Data type

A date time. The value space of :doc:`DateTime </reference/DataTypes/DateTime>` is the same as that of the XML Schema data type `dateTime <https://www.w3.org/TR/xmlschema-2/#dateTime>`__ as specified by the W3C Recommendation `XML Schema Part 2: Datatypes Second Edition <https://www.w3.org/TR/xmlschema-2>`__ from October 28, 2004, with the exception that :doc:`DateTime </reference/DataTypes/DateTime>` values do not carry any timezone information.

.. rubric:: Supertypes

* :doc:`NullableDateTime </reference/DataTypes/NullableDateTime>`

.. admonition:: proteus

   The base type :doc:`NullableDateTime </reference/DataTypes/NullableDateTime>` is only used as the type of the :ref:`ExportDateTime <dexpi-DexpiModel-DexpiModel-ExportDateTime>` attribute of :doc:`DexpiModel </reference/DexpiModel/DexpiModel>`. For this attribute, special rules apply; see Proteus Schema implementation of :ref:`ExportDateTime <dexpi-DexpiModel-DexpiModel-ExportDateTime>`. There are no attributes with type :doc:`DateTime </reference/DataTypes/DateTime>`.
