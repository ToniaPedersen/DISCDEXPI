==============
NullableString
==============

Overview
========

.. rubric:: Abstract data type

A sequence of characters, or a null value. :doc:`NullableString </reference/DataTypes/NullableString>` is abstract and has two concrete subtypes:

* a :doc:`String </reference/DataTypes/String>` is an :ref:`actual value <dexpi-concept-actual-value>` for a sequence of characters;

* a :doc:`NullString </reference/DataTypes/NullString>` is a :ref:`null value <dexpi-concept-null-value>` that explicitly indicates the absence of a sequence of characters.

.. image:: NullableString.*


.. rubric:: Subtypes

* :doc:`NullString </reference/DataTypes/NullString>`

* :doc:`String </reference/DataTypes/String>`

.. admonition:: proteus

   The Proteus Schema implementation of :doc:`NullableString </reference/DataTypes/NullableString>` depends on the Proteus Schema implementations of the data attributes with type :doc:`NullableString </reference/DataTypes/NullableString>`.

   * :doc:`NullableString </reference/DataTypes/NullableString>` is the type of the :ref:`Value <dexpi-Customization-CustomStringAttribute-Value>` attribute of :doc:`CustomStringAttribute </reference/Customization/CustomStringAttribute>`. :doc:`CustomStringAttribute </reference/Customization/CustomStringAttribute>` is implemented as a :ref:`DEXPI custom generic attribute <proteus-concept-dexpi-custom-generic-attribute>`.

   * All other data attributes with type :doc:`NullableString </reference/DataTypes/NullableString>` are implemented as :ref:`DEXPI generic attributes <proteus-concept-dexpi-generic-attribute>`. For example, see the :ref:`MaterialOfConstructionCode <dexpi-Equipment-AgitatorRotor-MaterialOfConstructionCode>` attribute of :doc:`AgitatorRotor </reference/Equipment/AgitatorRotor>`.

   Note that there are further data attributes with type :doc:`String </reference/DataTypes/String>` instead of :doc:`NullableString </reference/DataTypes/NullableString>`. See the Proteus Schema implementation of :doc:`String </reference/DataTypes/String>` for a list of these special cases.
