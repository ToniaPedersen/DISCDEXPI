.. _dexpi-Instrumentation-ActuatingFunction-ActuatingFunctionNumber:

ActuatingFunctionNumber
=======================

.. rubric:: Attribute (data)


An identifier for the :doc:`ActuatingFunction </reference/Instrumentation/ActuatingFunction>`. It usually contains the identifier of the :doc:`ProcessInstrumentationFunction </reference/Instrumentation/ProcessInstrumentationFunction>` that includes the :doc:`ActuatingFunction </reference/Instrumentation/ActuatingFunction>` (see :ref:`ProcessInstrumentationFunctionNumber <dexpi-Instrumentation-ProcessInstrumentationFunction-ProcessInstrumentationFunctionNumber>`\ ).

Multiplicity
    0..1

Type
    :doc:`NullableString </reference/DataTypes/NullableString>`

.. admonition:: proteus

   The attribute is implemented as a :ref:`DEXPI generic attribute for string values <proteus-concept-dexpi-generic-attribute-string>`.

   RDL reference
       `ACTUATING FUNCTION NUMBER ASSIGNMENT CLASS <http://sandbox.dexpi.org/rdl/ActuatingFunctionNumberAssignmentClass>`__

   Name
       ActuatingFunctionNumberAssignmentClass

   AttributeURI
       `http://sandbox.dexpi.org/rdl/ActuatingFunctionNumberAssignmentClass <http://sandbox.dexpi.org/rdl/ActuatingFunctionNumberAssignmentClass>`__

.. admonition:: example

   "HV4750.01" (:doc:`String </reference/DataTypes/String>`\ )

   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <ActuatingFunction
            ID="actuatingFunction1"
            ComponentClass="ActuatingFunction"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/ActuatingFunction" DOTS123="DOTS123">
          ...
          <GenericAttributes Set="DexpiAttributes" DOTS123="DOTS123">
            <GenericAttribute
              Name="ActuatingFunctionNumberAssignmentClass"
              AttributeURI="http://sandbox.dexpi.org/rdl/ActuatingFunctionNumberAssignmentClass"
              Format="string"
              Value="HV4750.01" />
            ...
          </GenericAttributes>
          ...
        </ActuatingFunction>
