.. admonition:: proteus

   :doc:`VolumeFlowRateUnit </reference/PhysicalQuantities/VolumeFlowRateUnit>` is only used as the type of the :ref:`Unit <dexpi-PhysicalQuantities-VolumeFlowRate-Unit>` attribute of :doc:`VolumeFlowRate </reference/PhysicalQuantities/VolumeFlowRate>`. :doc:`VolumeFlowRate </reference/PhysicalQuantities/VolumeFlowRate>` is implemented using :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`. In a ``<GenericAttribute>`` element, the :doc:`VolumeFlowRateUnit </reference/PhysicalQuantities/VolumeFlowRateUnit>` literal is given by means of its :ref:`RDL reference <proteus-concept-rdl-reference>` in the table above. The ``Units`` attribute of the element is the name of the RDL reference in camel-case; it equals the name of the literal. The ``UnitsURI`` attribute of the element is the URI of the RDL reference.
