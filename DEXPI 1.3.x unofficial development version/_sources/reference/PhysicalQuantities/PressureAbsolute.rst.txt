================
PressureAbsolute
================

Overview
========

.. rubric:: Data type

An :ref:`actual value <dexpi-concept-actual-value>` for a physical quantity of type :doc:`NullablePressureAbsolute </reference/PhysicalQuantities/NullablePressureAbsolute>`, i.e., a physical quantity that has a numerical value and a unit of measurement.

| :ref:`Value <dexpi-PhysicalQuantities-PressureAbsolute-Value>`                             | 1            | :doc:`Double </reference/DataTypes/Double>`                                                                        |

.. admonition:: proteus

   All data attributes with type :doc:`NullablePressureAbsolute </reference/PhysicalQuantities/NullablePressureAbsolute>` (the base type of :doc:`PressureAbsolute </reference/PhysicalQuantities/PressureAbsolute>`\ ) are implemented as :ref:`DEXPI generic attributes for physical quantities <proteus-concept-dexpi-generic-attribute-physical-quantity>`.

.. admonition:: example

   The instance pressureAbsolute1 represents a :doc:`PressureAbsolute </reference/PhysicalQuantities/PressureAbsolute>` of 4.8 bar.

   .. image:: /diagrams/diagram611.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <GenericAttribute
          Format="double"
          Value="4.8"
          Units="Bar"
          UnitsURI="http://data.posccaesar.org/rdl/RDS1314539" DOTS123="DOTS123">
      

      Note that the ``<GenericAttribute>`` element must have a ``Name`` and an ``AttributeURI`` attribute. They depend on the data type attribute of the DEXPI class that owns the instance pressureAbsolute1. For a complete example, see the attribute :ref:`DesignDifferentialPressure <dexpi-Equipment-Blower-DesignDifferentialPressure>` of the DEXPI class :doc:`Blower </reference/Equipment/Blower>`.
